# learn

Interactive learning tools for understanding your codebase - guided tours, explanations, patterns, and glossary.

## Subcommands

| Command | Description |
|---------|-------------|
| `/agileflow:learn:tour` | Guided codebase walkthrough with annotated stops |
| `/agileflow:learn:explain` | Explain a specific file, function, or concept |
| `/agileflow:learn:patterns` | Identify and explain design patterns in use |
| `/agileflow:learn:glossary` | Project-specific terminology and concepts |

## Quick Start

```
/agileflow:learn:tour                            # Start a guided tour
/agileflow:learn:explain src/auth/               # Explain the auth module
/agileflow:learn:patterns                        # Show patterns in use
/agileflow:learn:glossary                        # Browse project glossary
```

Great for onboarding new team members or understanding unfamiliar parts of the codebase.
