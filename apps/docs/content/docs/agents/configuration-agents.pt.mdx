# Agentes de Configuração

Os Agentes de Configuração fornecem setup automatizado, descoberta de capacidade de IDE, gerenciamento de estado de sessão e coordenação entre múltiplos agentes.

## Capacidades

- **Descoberta de IDE**: Detectar IDEs instaladas (Claude Code, Cursor, Windsurf)
- **Gerenciamento de Estado de Sessão**: Rastrear contexto através de conversas
- **Armazenamento de Arquivo de Projeto**: Salvar arquivos específicos de projeto
- **Configuração de Hook**: Setup de SessionStart, BeforeCommit, AfterCommit hooks
