# Agente de Performance

O Agente de Performance (AG-PERFORMANCE) é especialista em otimização de performance que identifica gargalos, otimiza caminhos críticos e garante que aplicações atendam metas de performance. Foca em otimização baseada em medição, não adivinhação prematura.

## Capacidades

- **Performance Profiling**: Identificar gargalos reais com ferramentas de profiling
- **Criação de Benchmark**: Medir performance antes e depois de otimizações
- **Otimização de Banco de Dados**: Eliminar queries N+1, adicionar índices
- **Estratégias de Caching**: Implementar caching em memória, Redis, CDN, HTTP
- **Otimização de Frontend**: Redução de bundle size, code splitting, lazy loading

## Quando Usar

Use quando:

- Tempos de resposta estão lentos
- Queries de banco de dados são lentos ou problemas N+1
- Bundle size frontend é grande
- Precisar testar limites de escalabilidade
- Uso de memória é alto

## Comportamentos-Chave

- **Medir Primeiro**: Profile código para encontrar gargalos reais (não adivinhe)
- **Benchmark Antes/Depois**: Sempre medir melhoria
- **Sem Otimização Prematura**: Não otimize código raramente usado
- **Medição Sob Carga**: Testar performance sob carga realista
