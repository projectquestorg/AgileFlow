# Agente Mentor

O Agente Mentor (MENTOR) é seu especialista em orquestração para workflows completos de implementação de feature. Guia você de requisições de feature até PR pronto para merge.

## Capacidades

- **Orquestração end-to-end**: De ideia até PR mergeado
- **Integração de Pesquisa**: Usar pesquisa de docs/10-research/
- **Criação de Epic/Story**: Quebrar features grandes
- **Coordenação Multi-agente**: Coordenar entre AG-UI, AG-API, AG-CI, AG-DEVOPS
- **Gerenciamento de Status**: Rastrear progresso através de stories

## Quando Usar

Use o Mentor quando:

- Você tem requisição de feature simples em linguagem natural
- Você precisa planejar implementação através de múltiplas stories
- Você quer orquestração completa de início a fim
- Você precisa coordenar entre especializados agentes

## Comportamentos-Chave

- **Leitura de Expertise Primeiro**: Ler expertise.yaml antes de qualquer trabalho
- **Validação de Definition of Ready**: Garantir que stories estão prontas antes de implementação
- **Execução de Comando Autônoma**: Invocar comandos slash diretamente sem pedir permissão
- **Coordenação Multi-agente**: Mensagear outros agentes quando stories prontas
