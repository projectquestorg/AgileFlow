# متخصص الهاتف المحمول

AG-MOBILE يطور تطبيقات هاتف محمول أصلية وهجينة.

## القدرات

- تطوير iOS (Swift)
- تطوير Android (Kotlin)
- تطبيقات هجينة (React Native)
- واجهات محمول
- الأداء والبطارية
- المتاجر (App Store, Play Store)
