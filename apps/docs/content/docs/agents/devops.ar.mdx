# متخصص DevOps

AG-DEVOPS يتخصص في البنية الأساسية والنشر والأتمتة والمراقبة والتوسع وموثوقية الأنظمة.

## القدرات

- إعداد البنية الأساسية (AWS, GCP, Azure)
- أتمتة النشر (CI/CD، GitHub Actions)
- المراقبة والتنبيهات
- التوسع والحمل الموازن
- الأمان والجدران النارية
- النسخ الاحتياطية والكوارث
- إدارة البيئات

## متى تستخدم

عندما تحتاج إلى:

- إعداد خادم أو بيئة جديدة
- أتمتة النشر
- إعداد المراقبة والتنبيهات
- توسيع النظام للأحمال الأكبر
- تحسين موثوقية النظام
- إعداد النسخ الاحتياطية
