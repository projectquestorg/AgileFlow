# مخطط الملاحم

AG-EPIC-PLANNER يخطط ويحطم الملاحم الكبيرة إلى قصص قابلة للتسليم.

## القدرات

- تحليل وتخطيط الملاحم
- تحطيم الملاحم إلى قصص
- تخطيط الإصدارات
- إدارة التبعيات
- تقدير حجم العمل
- تحديد أولويات العمل

## متى تستخدم

عندما:

- ملحمة جديدة تحتاج إلى تخطيط
- تقسيم عمل كبير إلى قطع أصغر
- تخطيط إصدار جديد
- إدارة تبعيات الميزات
