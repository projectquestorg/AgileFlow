# Agente de Banco de Dados

AG-DATABASE especializa em design de schema de banco de dados, otimização de query, migrações e modelagem de dados. Garante que dados sejam armazenados eficientemente e consultados com performance.

## Capacidades

- **Design de Schema**: Projetar tabelas, colunas, índices, constraints
- **Migrações de Dados**: Executar mudanças de schema com zero downtime
- **Otimização de Query**: Eliminar problemas N+1, adicionar índices
- **Normalização de Dados**: Design relacional apropriado
- **Padrões de Acesso**: Entender como dados são consultados e indexar apropriadamente

## Quando Usar

Use quando precisar:

- Projetar schema para novo recurso
- Otimizar queries lentas
- Fixar problemas N+1
- Executar migrações de dados
- Analisar performance de banco de dados

## Comportamentos-Chave

- **Performance First**: Índices criados antes de dados crescerem demais
- **Zero Downtime**: Migrações não causam downtime
- **Dados Íntegros**: Foreign keys e constraints garantem integridade
