# AgileFlow

AgileFlow é uma **estrutura de gerenciamento de projetos baseada em IA** projetada especificamente para [Claude Code__NOTRASLATE_44__. Ele traz fluxos de trabalho ágeis estruturados, agentes inteligentes e gerenciamento de projetos de documentos como código diretamente para o seu ambiente de desenvolvimento.

## Por que AgileFlow?

As ferramentas tradicionais de gerenciamento de projetos criam atrito entre o planejamento e a execução. AgileFlow elimina essa lacuna incorporando o gerenciamento de projetos diretamente em seu fluxo de trabalho de codificação assistido por IA.

- **Sem mudança de contexto** - Gerencie épicos, histórias e status sem sair do seu terminal
- **Fluxos de trabalho nativos de IA** - Desenvolvido especificamente para os recursos do Claude Code
- **Docs-as-code** - Todos os artefatos do projeto residem em seu repositório
- **Agentes inteligentes** - Agentes de IA especializados para diferentes domínios

## Principais recursos

### 74 comandos de barra

AgileFlow fornece uma biblioteca de comandos abrangente para todos os aspectos do gerenciamento de projetos:

| Categoria | Comandos |
|----------|----------|
| **Planejamento** | `/epic`, `/story`, `/sprint`, `/deps` |
| **Rastreamento** | `/status`, `/board`, `/blockers`, `/metrics` |
| **Desenvolvimento** | `/babysit`, `/verify`, `/review`, `/pr` |
| **Operações** | `/ci`, `/deploy`, `/changelog`, `/packages` |
| **Pesquisa** | `/research:ask`, `/research:import`, `/context:full`, `/context:export`, `/multi-expert` |

### 30 Agentes Especializados

Cada agente é um especialista em sua área, pronto para ajudar em tarefas específicas:

- **mentor** - Orientação de implementação ponta a ponta
- **epic-planner** - Divida os recursos em histórias
- **banco de dados** - Design de esquema e migrações
- **api** - Serviços de back-end e camada de dados
- **ui** - Componentes e estilo de front-end
- **segurança** - Análise de vulnerabilidades e conformidade
- **testes** - Estratégia e cobertura de teste
- **devops** - CI/CD e implantação

## Exemplo rápido

Esta é a aparência de um fluxo de trabalho típico do AgileFlow:

```bash
# 1. Create an epic for a new feature
/epic EPIC=EP-0001 TITLE="User Authentication" GOAL="Secure user login system"

# 2. Break it down into stories
/story EPIC=EP-0001 STORY=US-0001 TITLE="Login form" OWNER=AG-UI

# 3. Start working on a story
/status STORY=US-0001 STATUS=in-progress

# 4. Get AI-guided implementation
/babysit

# 5. Verify your implementation
/verify US-0001

# 6. Generate a PR
/pr STORY=US-0001
```

## Como funciona

1. **Instalar AgileFlow** – Um comando adiciona todos os recursos ao seu projeto
2. **Planejar com IA** – Use agentes para dividir recursos em histórias acionáveis
3. **Acompanhe o progresso** - Atualizações de status em tempo real em __NOTRASLATE_23__
4. **Colaborar** – Os agentes se comunicam por meio do barramento de mensagens
5. **Enviar** - Geração automatizada de PR e changelogatualizações

## Filosofia

AgileFlow é construído sobre estes princípios:

- **Convenção sobre configuração** - Padrões sensatos que simplesmente funcionam
- **Transparência** - Todos os dados são em texto simples, com versão controlada
- **Composibilidade** - Use apenas o que você precisa
- **AI-first** - Projetado para colaboração entre humanos e IA
- **Internacionalização** - Documentação disponível em 5 idiomas (espanhol, francês, alemão, português, árabe)
