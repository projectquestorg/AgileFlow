# AgileFlow

AgileFlow ist ein **AI-powered project management framework** speziell für konzipiert [Claude Code](https://claude.ai/code). Es bringt strukturierte, agile Arbeitsabläufe, intelligente Agenten und Docs-as-Code-Projektmanagement direkt in Ihre Entwicklungsumgebung.

## Warum AgileFlow?

Herkömmliche Projektmanagement-Tools erzeugen Reibungen zwischen Planung und Ausführung. AgileFlow schließt diese Lücke, indem es das Projektmanagement direkt in Ihren KI-gestützten Codierungsworkflow einbettet.

- **No context switching** - Verwalten Sie Epics, Storys und Status, ohne Ihr Terminal zu verlassen
- **AI-native workflows** - Speziell für die Fähigkeiten von Claude Code entwickelt
- **Docs-as-code** - Alle Projektartefakte befinden sich in Ihrem Repository
- **Intelligent agents** - Spezialisierte KI-Agenten für verschiedene Domänen

## Hauptmerkmale

### 74 Slash-Befehle

AgileFlow bietet eine umfassende Befehlsbibliothek für jeden Aspekt des Projektmanagements:

| Category | Commands |
|----------|----------|
| **Planning** | `/epic`, `/story`, `/sprint`, `/deps` |
| **Tracking** | `/status`, `/board`, `/blockers`, `/metrics` |
| **Development** | `/babysit`, `/verify`, `/review`, `/pr` |
| **Operations** | `/ci`, `/deploy`, `/changelog`, `/packages` |
| **Research** | `/research:ask`, `/research:import`, `/context:full`, `/context:export`, `/multi-expert` |

### 30 spezialisierte Agenten

Jeder Agent ist ein Experte auf seinem Gebiet und bereit, bei bestimmten Aufgaben zu helfen:

- **mentor** - End-to-End-Implementierungsanleitung
- **epic-planner** - Teilen Sie Features in Geschichten auf
- **database** - Schemadesign und Migrationen
- **api** - Backend-Dienste und Datenschicht
- **ui** - Frontend-Komponenten und -Styling
- **security** - Schwachstellenanalyse und Compliance
- **testing** - Teststrategie und -abdeckung
- **devops** - CI/CD und Bereitstellung

## Kurzes Beispiel

So sieht ein typischer AgileFlow-Workflow aus:

```bash
# 1. Create an epic for a new feature
/epic EPIC=EP-0001 TITLE="User Authentication" GOAL="Secure user login system"

# 2. Break it down into stories
/story EPIC=EP-0001 STORY=US-0001 TITLE="Login form" OWNER=AG-UI

# 3. Start working on a story
/status STORY=US-0001 STATUS=in-progress

# 4. Get AI-guided implementation
/babysit

# 5. Verify your implementation
/verify US-0001

# 6. Generate a PR
/pr STORY=US-0001
```

## Wie es funktioniert

1. **Install AgileFlow** - Ein Befehl fügt Ihrem Projekt alle Funktionen hinzu
2. **Plan with AI** - Verwenden Sie Agenten, um Funktionen in umsetzbare Geschichten aufzuschlüsseln
3. **Track Progress** - Statusaktualisierungen in Echtzeit `status.json`
4. **Collaborate** - Agenten kommunizieren über den Nachrichtenbus
5. **Ship** - Automatisierte PR-Generierung und Änderungsprotokollaktualisierungen

## Philosophie

AgileFlow basiert auf diesen Prinzipien:

- **Convention over configuration** - Sinnvolle Standardeinstellungen, die einfach funktionieren
- **Transparency** - Alle Daten sind Klartext und versioniert
- **Composability** - Verwenden Sie nur das, was Sie brauchen
- **AI-first** - Entwickelt für die Zusammenarbeit zwischen Mensch und KI
- **Internationalization** - Dokumentation in 5 Sprachen verfügbar (Spanisch, Französisch, Deutsch, Portugiesisch, Arabisch)
