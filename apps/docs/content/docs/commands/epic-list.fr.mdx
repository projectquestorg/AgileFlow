# /epic:liste

Affichez toutes les épopées avec le statut, le suivi de la progression et les options d'action rapide.

## Démarrage rapide

```bash
/agileflow:epic:list
```

## Paramètres

| Paramètre | Obligatoire | Par défaut | Descriptif |
|-----------|----------|---------|-------------|
| `STATUS` | Non | - | Filtrer par statut : `active`, `complete`, `on-hold` |

## Exemples

### Liste toutes les épopées

```bash
/agileflow:epic:list
```

Affiche toutes les épopées avec leur statut et leur pourcentage d'achèvement de l'histoire.

### Répertorier uniquement les épopées actives

```bash
/agileflow:epic:list STATUS=active
```

Filtres pour afficher uniquement les épopées en cours.

### Liste uniquement les épopées terminées

```bash
/agileflow:epic:list STATUS=complete
```

Affiche les épopées dont toutes les histoires sont terminées.

## Format de sortie

La commande affiche les epics dans un tableau structuré :

```markdown
## Epics

| Epic | Title | Status | Progress | Owner |
|------|-------|--------|----------|-------|
| EP-0003 | User Dashboard | active | 2/5 (40%) | AG-UI |
| EP-0002 | Authentication | active | 4/6 (67%) | AG-API |
| EP-0001 | Core Setup | complete | 3/3 (100%) | AG-DEVOPS |

---
**Summary**: 3 epics (2 active, 1 complete)
**Stories**: 9/14 complete (64%)
```

## Définitions de statut

| Statut | Signification | Barre de progression | Actions |
|--------|---------|------|--------|
| **actif** | En cours, acceptation de nouvelles histoires | Remplissage partiel | Continuer le travail, ajouter des histoires |
| **complète** | Toutes les histoires terminées | Complet (100%) | Voir le résumé, éventuellement rouvrir |
| **en attente** | En pause, pas de travail actuellement | Congelé | Reprendre ou fermer |

## Visualisation des progrès

Les progrès sont indiqués comme suit :
- **Numérateur** : histoires terminées
- **Dénominateur** : Total des histoires en épopée
- **Pourcentage** : achèvement global

Exemple : `4/6 (67%)` = 4 histoires terminées, 2 restantes, terminées à 67 %

## Actions rapides

Après avoir affiché la liste, vous pouvez :

### Afficher les détails épiques

Sélectionnez une épopée pour voir :
- Toutes les histoires regroupées par statut (in_progress, ready, done)
- Visualisation des progrès
- Propriétaire et objectif
- Recherches connexes et ADR

```bash
/agileflow:epic:view EPIC=EP-0003
```

### Ajouter une histoire à Epic

Créez une nouvelle histoire sous une épopée existante :

```bash
/agileflow:story EPIC=EP-0003 ...
```

### Créer une nouvelle épopée

Démarrez une nouvelle épopée de fonctionnalités :

```bash
/agileflow:epic TITLE="..." ...
```

## Intégration du flux de travail

### Planification des sprints

```bash
/agileflow:epic:list STATUS=active
# See which epics are in progress
/agileflow:epic:view EPIC=EP-0003
# Review stories and their status
/agileflow:sprint
# Plan which stories to work on
```

### Planification trimestrielle

```bash
/agileflow:epic:list
# See all epics and their progress
# Identify bottlenecks (epics with less than 50% progress)
# Decide which epics to prioritize next
```

### Suivi de l'achèvement

```bash
/agileflow:epic:list STATUS=active
# Work on remaining stories in in-progress epics
/agileflow:epic:list STATUS=complete
# Review what was shipped
```

### Épopées en attente

```bash
/agileflow:epic:list STATUS=on-hold
# Check if any on-hold epics can be resumed
/agileflow:epic:view EPIC=EP-0002
# Understand why it's on hold
```

## Interprétation des résultats

### Progrès élevés (supérieurs à 80 %)

```
EP-0002 | Authentication | active | 5/6 (83%) | AG-API
```

- Epic est presque terminé
- Concentrez-vous sur la fin de la dernière histoire
- Planifiez la prochaine épopée pour commencer
- Ne fermez pas avant 100%

### Progrès moyen (40-80 %)

```
EP-0003 | User Dashboard | active | 3/7 (43%) | AG-UI
```

- Epic est en plein sprint
- Peut avoir besoin d'histoires supplémentaires
- Vérifiez si les histoires sont bloquées
- Pensez à ajouter des ressources

### Faibles progrès (inférieurs à 40 %)

```
EP-0001 | Onboarding Flow | active | 1/8 (12%) | AG-UI
```

- Epic vient de démarrer
- Montée en puissance en cours
- Surveiller les bloqueurs
- Vous voudrez peut-être ajouter plus d'histoires
