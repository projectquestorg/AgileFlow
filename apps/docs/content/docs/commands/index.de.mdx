## Nutzungsmuster

### Grundlegende Verwendung

```bash
# Simple command
/agileflow:board

# Command with required parameters
/agileflow:status STORY=US-0001 STATUS=in-progress

# Command with optional parameters
/agileflow:metrics TIMEFRAME=30d EPIC=EP-0001 FORMAT=markdown
```

### Parameterkonventionen

- **Required parameters**: `PARAM=<value>` - Muss bereitgestellt werden
- **Optional parameters**: `[PARAM=<value>]` - Kann für Standardwerte weggelassen werden
- **IDs**: Verwenden Sie Muster wie `EP-0001`, `US-0001`, `AG-UI`
- **Lists**: Durch Kommas getrennte Werte, sofern angegeben

### Hilfe bekommen

```bash
# Show all available commands
/agileflow:help

# System diagnostics
/agileflow:diagnose
```
