# /idée:histoire

Interrogez les rapports d'idéation antérieurs, suivez l'état des idées, comparez les rapports au fil du temps et identifiez les thèmes récurrents.

## Démarrage rapide

```bash
/agileflow:ideate:history
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[IDEA-XXXX]` | View specific idea by ID |
| `STATUS=pending\|in-progress\|implemented\|recurring` | Filter by status |
| `MODE=trends` | Show aggregate statistics across all reports |
| `COMPARE=report1,report2` | Diff two reports |

## Caractéristiques

- **Status filtering** - Trouver des idées par leur statut actuel
- **Recurring detection** - Met en évidence les idées apparues dans plus de 2 rapports
- **Trend analysis** - Regrouper les statistiques dans tous les rapports d'idéation
- **Report comparison** - Diff deux rapports affichant Résolu/Nouveau/Persisté/Abandonné

## Exemples

```bash
# View all pending ideas
/agileflow:ideate:history STATUS=pending

# Check trends across reports
/agileflow:ideate:history MODE=trends

# Compare two reports
/agileflow:ideate:history COMPARE=ideation-20260101.md,ideation-20260201.md
```
