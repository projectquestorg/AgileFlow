# /الجلسة:الحالة

عرض جدول منسق بجميع الجلسات المسجلة، موضحًا الجلسات النشطة حاليًا.

## بداية سريعة

__لا تترجم_0__

## إخراج المثال

__لا تترجم_1__

## قيم الحالة

| الحالة | معنى |
|--------|---------|
| __غير مترجم_5__ | تحتوي الجلسة على عملية Claude Code قيد التشغيل |
| __غير مترجم_6__ | ملف القفل موجود ولكن العملية لم تعد قيد التشغيل |
| __غير مترجم_7__ | دليل المشروع الأصلي (الجلسة 1) |

## كشف الجلسة

يتم اكتشاف الجلسات باستخدام القفل المستند إلى PID:

1. تقوم كل جلسة بإنشاء ملف قفل بمعرف العملية الخاص به
2. يتحقق فحص الحالة مما إذا كان كل معرف PID لا يزال قيد التشغيل
3. يتم تنظيف الأقفال القديمة تلقائيًا عند بدء التشغيل التالي

## حالات الاستخدام

### قبل البدء بالعمل

تحقق مما إذا كانت الجلسات الأخرى تعمل بالفعل على شيء ما:

__لا تترجم_2__

### بعد العودة إلى العمل

تعرف على الجلسات الموجودة وحالتها:

__لا تترجم_3__

### تصحيح التعارضات

إذا كنت تشك في حدوث تعارضات في الجلسة:

__لا تترجم_4__

## الفاصلة ذات الصلةnds

- [`/session:new`](/commands/session-new) - إنشاء جلسة موازية جديدة
- [`/session:resume`](/commands/session-resume) - التبديل إلى جلسة أخرى
- [`/session:end`](/commands/session-end) - إنهاء الجلسة الحالية
- [`/session:init`](/commands/session-init) - تهيئة نظام الجلسة
