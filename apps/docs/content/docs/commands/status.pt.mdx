# /status

Atualize o status da história no sistema de rastreamento de projetos e transmita as alterações aos agentes por meio do barramento de mensagens.

## Início rápido

```bash
/agileflow:status STORY=US-0001 STATUS=in-progress
```

## Parâmetros

| Parâmetro | Obrigatório | Padrão | Descrição |
|-----------|----------|---------|------------|
| `STORY` | Sim | - | ID da história (por exemplo, US-0001, US-0042) |
| `STATUS` | Sim | - | Novo status: `ready`, `in-progress`, `blocked`, `in-review` ou `done` |
| `SUMMARY` | Não | - | Resumo de 1-2 linhas do progresso ou motivo da mudança de status |
| `PR` | Não | - | URL de solicitação pull (para status de revisão/concluído) |
| `TO` | Não | TODOS | Agente(s) alvo a notificar através do barramento de mensagens |

## Exemplos

### Comece a trabalhar em uma história

```bash
/agileflow:status STORY=US-0042 STATUS=in-progress SUMMARY="Beginning login form implementation"
```

Isso atualiza o status da história e notifica todos os agentes de que você começou a trabalhar.

### Marcar como bloqueado

```bash
/agileflow:status STORY=US-0045 STATUS=blocked SUMMARY="Waiting on AG-API to implement /auth/login endpoint"
```

Transmita o bloqueador para que outros membros da equipe possam ajudar a resolvê-lo.

## Valores de status explicados

- **pronto** - A história está preparada e esperando para ser iniciada
- **em andamento** - Atualmente sendo trabalhado ativamente
- **bloqueado** - Aguardando dependência externa ou esclarecimento
- **em revisão** - Concluído e aguardando revisão/aprovação do código
- **concluído** - Concluído, testado e verificado

## Melhores práticas

1. **Sempre inclua um resumo** - Ajuda a equipe a entender o contexto e o progresso2. **Atualize imediatamente** - Não espere atualizações de status em lote
3. **Vincular PRs para avaliações** – Ajuda os revisores a encontrar seu trabalho
4. **Use uma linguagem clara** - Explique os bloqueadores em detalhes para que outras pessoas possam ajudar
5. **Atualizar quando bloqueado** - Não fique sentado silenciosamente nos bloqueadores - comunique-se imediatamente
