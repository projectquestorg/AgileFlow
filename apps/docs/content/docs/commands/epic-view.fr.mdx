# /epic:vue

Affichez tous les détails de l'Epic, y compris toutes les histoires regroupées par statut et actions contextuelles.

## Démarrage rapide

```bash
/agileflow:epic:view EPIC=EP-0002
```

## Paramètres

| Parameter | Required | Default | Description |
|-----------|----------|---------|-------------|
| `EPIC` | Yes | - | Epic ID (e.g., EP-0001) |

## Exemples

### Voir l'épopée spécifique

```bash
/agileflow:epic:view EPIC=EP-0002
```

Affiche des informations épiques complètes avec toutes les histoires et actions contextuelles.

### Sélection interactive

```bash
/agileflow:epic:view
```

Sans paramètre EPIC, affiche les épopées récentes et vous demande d'en sélectionner une.

## Actions contextuelles

Les actions changent en fonction de la progression épique :

### Pour les épopées à faible progression (inférieure à 50 %)

Lorsque la progression est inférieure à 50 % :

```
This epic is 40% complete. What would you like to do?
```

Possibilités :
- **Add more stories**: Epic peut avoir besoin d'éléments de travail supplémentaires
- **Start work on a ready story**: Commencer la mise en œuvre
- **Plan sprint for this epic**: Programmer des histoires
- **Back to epic list**: Retour à la liste complète

### Pour les épopées à progression moyenne (50-99 %)

Lorsque la progression est comprise entre 50 % et 99 % :

```
This epic is 67% complete. What would you like to do?
```

Possibilités :
- **Work on remaining stories** (Recommandé) : Concentrez-vous sur la réalisation
- **View story details**: Vérifiez une histoire spécifique
- **Add more stories**: La portée épique a peut-être augmenté
- **Mark epic as complete**: Si les travaux restants sont différés

### Pour les épopées terminées (100 %)

Quand toutes les histoires sont terminées :

```
This epic is 100% complete! What would you like to do?
```

Possibilités :
- **Close epic** (Recommandé) : Marquer comme terminé dans status.json
- **View completion summary**: Voir ce qui a été livré
- **Add follow-up stories**: Epic a besoin de plus de travail après tout
- **Back to epic list**: Revenir à /agileflow:epic:list

### Pour les épopées en attente

Lorsque Epic est en pause :

```
This epic is on hold. What would you like to do?
```

Possibilités :
- **Resume epic**: Revenir à l'état actif
- **View why it's on hold**: Vérifiez les bloqueurs ou les notes
- **Close epic**: Plus nécessaire
- **Back to epic list**: Revenir à /agileflow:epic:list

## Flux de travail typiques

### Continuer à travailler sur Epic

```
1. /agileflow:epic:view EPIC=EP-0002
   (See 67% complete)
2. Select "Work on remaining stories"
3. See ready stories
4. Select a story
5. /agileflow:story:view STORY={selected}
6. Start implementation
```

### Terminer Epic à la fin du sprint

```
1. /agileflow:epic:view EPIC=EP-0002
   (See 95% complete)
2. Work through remaining stories
3. When all done:
4. Select "Close epic"
5. Epic marked as complete
6. Start next epic
```

### Débloquer l'Epic en attente

```
1. /agileflow:epic:view EPIC=EP-0003
   (Epic is on-hold)
2. Check blockers
3. If resolved: "Resume epic"
4. Epic changed back to active
5. Continue work
```

### Développez Epic avec plus de travail

```
1. /agileflow:epic:view EPIC=EP-0002
   (See 85% complete)
2. Realize more work is needed
3. Select "Add more stories"
4. /agileflow:story EPIC=EP-0002 ...
5. Create new stories
6. Update epic progress
```

## Signification du statut de l'histoire

Dans la vue épique, les histoires sont regroupées par statut :

| Status | Meaning | Next Action |
|--------|---------|-------------|
| **in_progress** | Currently being worked on | Continue work or mark as blocked |
| **ready** | Ready to start | Pick one and begin implementation |
| **done** | Completed and tested | Move to next epic |

## Interprétation des progrès

### Que signifie le pourcentage

```
4/6 (67%) = 4 stories done, 2 remaining, 67% complete
```

- 0-25 % : Epic vient de démarrer
- 25-50 % : montée en puissance, à mi-sprint
- 50-75 % : la plupart des travaux identifiés
- 75-100 % : dernière étape vers l'achèvement

### Quand ajouter plus d'histoires

Pensez à ajouter des histoires si :
- Les progrès plafonnent (bloqués au même pourcentage)
- L'équipe est bloquée en attente de dépendances
- De nouvelles exigences sont apparues

### Quand fermer Epic

Fermer quand :
- Toutes les histoires sont marquées comme `done`
- Les critères d'acceptation sont remplis
- Prêt à passer à la prochaine épopée
