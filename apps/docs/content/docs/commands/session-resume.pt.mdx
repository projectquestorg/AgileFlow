# /sessão:resume

Mude para uma sessão diferente do Claude Code. Mostra as sessões disponíveis e fornece o comando para alternar.

## Início rápido

```bash
/agileflow:session:resume
```

## Como funciona

1. Lista todas as sessões registradas com status
2. Apresenta seletor interativo para selecionar uma sessão
3. Mostra o comando do terminal para alternar

## Exemplo de interação

```
Which session do you want to resume?

○ Session 1 (main) - /home/user/myproject [active]
○ Session 2 "auth" - /home/user/myproject-auth [active]
○ Session 3 - /home/user/myproject-3 [stale]
```

Depois de selecionar a Sessão 2:

```
To resume Session 2 "auth":

  cd /home/user/myproject-auth && claude --resume

Note: Use a new terminal window/tab for best experience.
```

## Estados da Sessão

| Estado | Pode retomar? | Notas |
|-------|-------------|-------|
| `active` | Sim | Outra instância de Claude está rodando lá |
| `stale` | Sim | A sessão anterior terminou, o diretório ainda existe |
| `current` | Não | Você já está nesta sessão |

## O sinalizador --resume

A flag __NOTRASLATE_10__ diz ao Claude Code para:
- Carregar histórico de conversas da sessão anterior
- Restaure o contexto sobre o que você estava trabalhando
- Continue de onde parou

Sem __NOTRASLATE_11__, Claude começa com um novo contexto.
