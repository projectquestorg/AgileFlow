# /idear:historia

Consulte informes de ideas anteriores, realice un seguimiento del estado de las ideas, compare informes a lo largo del tiempo e identifique temas recurrentes.

## Inicio rápido

```bash
/agileflow:ideate:history
```

## Argumentos

| Argument | Description |
|----------|-------------|
| `[IDEA-XXXX]` | View specific idea by ID |
| `STATUS=pending\|in-progress\|implemented\|recurring` | Filter by status |
| `MODE=trends` | Show aggregate statistics across all reports |
| `COMPARE=report1,report2` | Diff two reports |

## Características

- **Status filtering** - Encuentra ideas por su estado actual.
- **Recurring detection** - Destaca ideas que aparecieron en más de 2 informes.
- **Trend analysis** - Estadísticas agregadas en todos los informes de ideación.
- **Report comparison** - Diferencia dos informes que muestran Resuelto/Nuevo/Persistente/Eliminado

## Ejemplos

```bash
# View all pending ideas
/agileflow:ideate:history STATUS=pending

# Check trends across reports
/agileflow:ideate:history MODE=trends

# Compare two reports
/agileflow:ideate:history COMPARE=ideation-20260101.md,ideation-20260201.md
```
