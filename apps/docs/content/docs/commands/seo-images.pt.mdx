# /seo:imagens

Analise a otimização de imagens em uma página: qualidade de texto alternativo, dimensionamento explícito para prevenção de CLS, formatos modernos (WebP/AVIF), carregamento lento, prioridade de imagem LCP e imagens responsivas.

## Início rápido

```bash
/agileflow:seo:images https://example.com                    # Image audit for homepage
/agileflow:seo:images https://example.com/products/item       # Product page images
```

## Parâmetros

| Parameter | Required | Description |
|-----------|----------|-------------|
| `<URL>` | Yes | Page to analyze |

| Aspect | Weight | Impact |
|--------|--------|--------|
| Alt text quality | 30% | Accessibility + image search rankings |
| Image sizing (CLS) | 20% | Core Web Vitals - layout stability |
| Modern formats | 15% | Page speed + LCP |
| Lazy loading | 15% | Page speed |
| LCP optimization | 10% | Core Web Vitals - largest paint |
| Responsive images | 10% | Mobile performance |

## Pontuação de texto alternativo

| Quality | Criteria | Score Impact |
|---------|----------|------------|
| **Good** | Descriptive, 10-125 chars, contextual keyword | +points |
| **Acceptable** | Present but generic ("image", "photo") | Neutral |
| **Poor** | Filename-based ("IMG_001.jpg", "screenshot") | -2 points |
| **Missing** | No alt attribute at all | -3 points (HIGH severity) |
| **Decorative** | Empty alt="" on decorative images | Correct (no deduction) |
| **Keyword-stuffed** | Excessive keywords crammed in | -2 points |

## Diretrizes de texto alternativo

Bom texto alternativo:
- Descreve o que a imagem mostra
- 10-125 caracteres recomendados
- Inclui palavras-chave relevantes naturalmente
- Não diz "imagem de" ou "imagem de"
- Para imagens decorativas, use `alt=""`

Exemplo de bom texto alternativo:
- "Mulher profissional digitando no laptop na mesa de escritório moderna"
- "Captura de tela do painel do Google Search Console mostrando métricas de tráfego"
- "Foto do produto: Fones de ouvido sem fio azuis com cancelamento de ruído"

Exemplo de texto alternativo ruim:
- "IMG_001.jpg" ❌
- "imagem" ❌
- "foto de coisas" ❌
- "blue-headphones-wireless-noise-cancelling-buy-now" (recheio de palavras-chave) ❌

## Recomendações de formato de imagem

| Format | Use When | Not When |
|--------|----------|----------|
| WebP | Modern format, wide support | Supporting very old browsers |
| AVIF | Best compression, growing support | Older browser support needed |
| JPEG | Photos, realistic images | Transparency needed |
| PNG | Transparency needed | Photos (use JPEG) |
| GIF | Animations | Use MP4 video instead |
| SVG | Icons, illustrations, logos | Photos |
| BMP/TIFF | Never | Never |

## Imagens responsivas

Para uma entrega ideal em dispositivos móveis e desktop:

```html
<!-- Good: Responsive with srcset and sizes -->
<img
  src="image-800.jpg"
  srcset="image-400.jpg 400w, image-800.jpg 800w, image-1200.jpg 1200w"
  sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px"
  width="800"
  height="600"
  alt="Descriptive alt text"
  loading="lazy"
>

<!-- Better: Format alternatives with <picture> -->
<picture>
  <source type="image/avif" srcset="image.avif">
  <source type="image/webp" srcset="image.webp">
  <img src="image.jpg" alt="Descriptive alt text" width="800" height="600">
</picture>
```

## Otimização de imagem LCP

Para o seu herói/imagem maior:

```html
<!-- Good: LCP image gets priority -->
<img
  src="hero.jpg"
  alt="Hero section describing main benefit"
  width="1200"
  height="600"
  fetchpriority="high"
  loading="eager"
/>

<!-- With preload for extra speed -->
<link rel="preload" as="image" href="hero.jpg">
<img src="hero.jpg" alt="..." width="1200" height="600">
```

## Remediation Examples

### Fix Missing Alt Text
```html
<!-- Antes -->
<img src="hero.jpg">

<!-- Depois -->
<img src="hero.jpg" alt="Team collaborating on project strategy in modern office">
```

### Fix Missing Width/Height
```html
<!-- Antes (risco CLS) -->
<img src="photo.jpg" alt="...">

<!-- Depois (evita mudança) -->
<img src="photo.jpg" alt="..." width="800" height="600">
```

### Add Lazy Loading
```html
<!-- Antes -->
<img src="below-fold.jpg" alt="...">

<!-- Depois -->
<img src="below-fold.jpg" alt="..." loading="lazy">
```

### Add Format Alternatives
```html
<!-- Antes -->
<img src="image.jpg" alt="..." width="800" height="600">

<!-- Depois (40% menor em navegadores modernos) -->
<picture>
  <source type="image/webp" srcset="image.webp">
  <img src="image.jpg" alt="..." width="800" height="600">
</picture>
```

### Add LCP Priority
```html
<!-- Antes -->
<img src="hero.jpg" alt="..." width="1200" height="600">

<!-- Depois (pintura mais rápida) -->
<img src="hero.jpg" alt="..." width="1200" height="600" fetchpriority="high">
```

## Impacto no desempenho

A otimização de imagens pode melhorar:
- **Core Web Vitals**
  - LCP: entrega mais rápida de imagens hero
  - CLS: Impede a mudança de layout de imagens não dimensionadas
- **Page Speed**: Carregamento 20-40% mais rápido com formatos modernos
- **User Experience**: Carregamento de página mais rápido, melhor desempenho percebido
- **SEO**: Melhor pontuação do Core Web Vitals → potencial aumento de classificação

## Resumo das melhores práticas

1. **Always include alt text** - Acessibilidade + pesquisa de imagens
2. **Always include width/height** - Impede mudança de layout (CLS)
3. **Use modern formats** - WebP para 30% menor, AVIF para 50%+
4. **Lazy load below-fold** - `loading="lazy"` adia imagens não críticas
5. **Prioritize LCP image** - `fetchpriority="high"` no herói
6. **Make responsive** - `srcset` e `sizes` para tamanhos apropriados ao dispositivo
7. **Test before publish** - Use o PageSpeed ​​​​Insights para verificar melhorias
