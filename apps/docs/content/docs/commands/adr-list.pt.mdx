# /adr:lista

Exiba todos os registros de decisões de arquitetura com status, datas e opções de ação rápida.

## Início rápido

```bash
/agileflow:adr:list
```

## Parâmetros

| Parâmetro | Obrigatório | Padrão | Descrição |
|-----------|----------|---------|------------|
| `STATUS` | Não | - | Filtrar por status: `proposed`, `accepted`, `deprecated`, `superseded` |

## Exemplos

### Listar todas as ADRs

```bash
/agileflow:adr:list
```

Mostra todos os ADRs classificados por número com seu status atual.

### Listar apenas ADRs aceitos

```bash
/agileflow:adr:list STATUS=accepted
```

Filtros para mostrar apenas ADRs que foram aprovadas e estão em vigor.

## Definições de status

| Estado | Significado | Ação |
|----|---------|--------|
| **proposta** | Em discussão, ainda não decidido | Revise e aprove ou solicite alterações |
| **aceitared** | Decisão tomada e em vigor | Implementar ou criar histórias relacionadas |
| **obsoleto** | Já não se aplica (mas não é substituído) | Considere reativar se o contexto mudar |
| **substituído** | Substituído por um ADR mais recente | Veja o ADR substituto para entender a evolução |
