# Comandos

## Padrões de uso

### Uso Básico

```bash
# Simple command
/agileflow:board

# Command with required parameters
/agileflow:status STORY=US-0001 STATUS=in-progress

# Command with optional parameters
/agileflow:metrics TIMEFRAME=30d EPIC=EP-0001 FORMAT=markdown
```

### Convenções de parâmetros

- **Parâmetros obrigatórios**: __NOTRASLATE_42__ - Devem ser fornecidos
- **Parâmetros opcionais**: __NOTRASLATE_43__ - Podem ser omitidos para padrões
- **IDs**: use padrões como `EP-0001`, `US-0001`, `AG-UI`
- **Listas**: valores separados por vírgula quando indicados

### Obtendo ajuda

```bash
# Show all available commands
/agileflow:help

# System diagnostics
/agileflow:diagnose
```
