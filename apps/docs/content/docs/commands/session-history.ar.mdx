## بيانات التاريخ

يتضمن سجل الجلسة ما يلي:

| المجال | الوصف |
|-------|------------|
| معرف الجلسة | المعرف الرقمي |
| اللقب | اسم صديق للإنسان إذا تم توفيره |
| المسار | موقع الدليل |
| فرع | فرع جيت مستعمل |
| الحالة | نشطة أو منتهية أو قديمة |
| تم الإنشاء | عندما بدأت الجلسة |
| انتهى | عند انتهاء الجلسة (إن أمكن) |
| المدة | كم من الوقت كانت الجلسة نشطة |
| شجرة العمل | سواء تم الاحتفاظ بشجرة العمل أو إزالتها |
