# /idear:história

Consulte relatórios de ideias anteriores, acompanhe o status das ideias, compare relatórios ao longo do tempo e identifique temas recorrentes.

## Início rápido

```bash
/agileflow:ideate:history
```

## Argumentos

| Argument | Description |
|----------|-------------|
| `[IDEA-XXXX]` | View specific idea by ID |
| `STATUS=pending\|in-progress\|implemented\|recurring` | Filter by status |
| `MODE=trends` | Show aggregate statistics across all reports |
| `COMPARE=report1,report2` | Diff two reports |

## Características

- **Status filtering** - Encontre ideias pelo status atual
- **Recurring detection** - Destaca ideias que apareceram em mais de 2 relatórios
- **Trend analysis** - Estatísticas agregadas em todos os relatórios de idealização
- **Report comparison** - Comparar dois relatórios mostrando Resolvido/Novo/Persistido/Descartado

## Exemplos

```bash
# View all pending ideas
/agileflow:ideate:history STATUS=pending

# Check trends across reports
/agileflow:ideate:history MODE=trends

# Compare two reports
/agileflow:ideate:history COMPARE=ideation-20260101.md,ideation-20260201.md
```
