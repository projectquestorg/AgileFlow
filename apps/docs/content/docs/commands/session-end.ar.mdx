# /الجلسة:النهاية

قم بإنهاء جلسة Claude Code بشكل نظيف، وإزالة قفلها وتنظيف شجرة عمل git بشكل اختياري.

## بداية سريعة

__لا تترجم_0__

## كيف يعمل

1. يعرض معلومات الجلسة الحالية
2. يسألك عما إذا كنت تريد إزالة شجرة عمل git
3. يزيل ملف قفل الجلسة
4. يقوم بحذف دليل شجرة العمل بشكل اختياري

## مثال للتفاعل

__لا تترجم_1__

## خيارات التنظيف

| الخيار | ماذا يحدث |
|--------|-------------|
| **احتفظ بشجرة العمل** | تمت إزالة قفل الجلسة، ويبقى الدليل. يمكنك `cd` هناك والعمل يدويًا. |
| **إزالة شجرة العمل** | تم حذف الدليل، وإزالة الفرع بشكل اختياري، والتنظيف الكامل. |

## متى تستخدم كل خيار

### حافظ على شجرة العمل

- تريد مواصلة العمل هناك بدون كلود
- أنت بحاجة إلى الحفاظ على التغييرات غير الملتزم بها
- ستعود إلى هذا العمل لاحقاً

### إزالة شجرة العمل

- تم الانتهاء من العمل ودمجه
- تريد تحرير مساحة القرص
- تنظيف التجارب المهجورة

### Git Worktree (اختياري)

## فحوصات السلامة

يمنع الأمر فقدان البيانات غير المقصود:

- **التغييرات غير الملتزم بها**: يظهر التحذير إذا كانت شجرة العمل تحتوي على عمل غير ملتزم به
- **الجلسة الرئيسية**: لا يمكن إزالة شجرة عمل المشروع الرئيسية
- **الجلسات النشطة**: تحذير في حالة محاولة إنهاء جلسة نشطة أخرى
