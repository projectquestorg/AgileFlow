# /seo:imagenes

Analice la optimización de imágenes en una página: calidad del texto alternativo, tamaño explícito para la prevención de CLS, formatos modernos (WebP/AVIF), carga diferida, prioridad de imagen LCP e imágenes responsivas.

## Inicio rápido

```bash
/agileflow:seo:images https://example.com                    # Image audit for homepage
/agileflow:seo:images https://example.com/products/item       # Product page images
```

## Parámetros

| Parameter | Required | Description |
|-----------|----------|-------------|
| `<URL>` | Yes | Page to analyze |

## Lo que se controla

| Aspect | Weight | Impact |
|--------|--------|--------|
| Alt text quality | 30% | Accessibility + image search rankings |
| Image sizing (CLS) | 20% | Core Web Vitals - layout stability |
| Modern formats | 15% | Page speed + LCP |
| Lazy loading | 15% | Page speed |
| LCP optimization | 10% | Core Web Vitals - largest paint |
| Responsive images | 10% | Mobile performance |

## Puntuación de texto alternativo

| Quality | Criteria | Score Impact |
|---------|----------|------------|
| **Good** | Descriptive, 10-125 chars, contextual keyword | +points |
| **Acceptable** | Present but generic ("image", "photo") | Neutral |
| **Poor** | Filename-based ("IMG_001.jpg", "screenshot") | -2 points |
| **Missing** | No alt attribute at all | -3 points (HIGH severity) |
| **Decorative** | Empty alt="" on decorative images | Correct (no deduction) |
| **Keyword-stuffed** | Excessive keywords crammed in | -2 points |

## Directrices de texto alternativo

Buen texto alternativo:
- Describe lo que muestra la imagen.
- Se recomiendan entre 10 y 125 caracteres
- Incluye palabras clave relevantes de forma natural
- No dice "imagen de" ni "imagen de"
- Para imágenes decorativas, utilice `alt=""`

Ejemplo de buen texto alternativo:
- "Mujer profesional escribiendo en una computadora portátil en el escritorio de la oficina moderna"
- "Captura de pantalla del panel de Google Search Console que muestra métricas de tráfico"
- "Foto del producto: Auriculares inalámbricos azules con cancelación de ruido"

Ejemplo de texto alternativo deficiente:
- "IMG_001.jpg" ❌
- "imagen" ❌
- "imagen de cosas" ❌
- "auriculares-azules-inalámbricos-con-cancelación-de-ruido-comprar-ahora" (relleno de palabras clave) ❌

## Recomendaciones de formato de imagen

| Format | Use When | Not When |
|--------|----------|----------|
| WebP | Modern format, wide support | Supporting very old browsers |
| AVIF | Best compression, growing support | Older browser support needed |
| JPEG | Photos, realistic images | Transparency needed |
| PNG | Transparency needed | Photos (use JPEG) |
| GIF | Animations | Use MP4 video instead |
| SVG | Icons, illustrations, logos | Photos |
| BMP/TIFF | Never | Never |

## Imágenes responsivas

Para una entrega óptima en dispositivos móviles y de escritorio:

```html
<!-- Good: Responsive with srcset and sizes -->
<img
  src="image-800.jpg"
  srcset="image-400.jpg 400w, image-800.jpg 800w, image-1200.jpg 1200w"
  sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px"
  width="800"
  height="600"
  alt="Descriptive alt text"
  loading="lazy"
>

<!-- Better: Format alternatives with <picture> -->
<picture>
  <source type="image/avif" srcset="image.avif">
  <source type="image/webp" srcset="image.webp">
  <img src="image.jpg" alt="Descriptive alt text" width="800" height="600">
</picture>
```

## Optimización de imagen LCP

Para tu héroe/imagen más grande:

```html
<!-- Good: LCP image gets priority -->
<img
  src="hero.jpg"
  alt="Hero section describing main benefit"
  width="1200"
  height="600"
  fetchpriority="high"
  loading="eager"
/>

<!-- With preload for extra speed -->
<link rel="preload" as="image" href="hero.jpg">
<img src="hero.jpg" alt="..." width="1200" height="600">
```

## Remediation Examples

### Fix Missing Alt Text
```html
<!-- Antes -->
<img src="hero.jpg">

<!-- Después -->
<img src="hero.jpg" alt="Team collaborating on project strategy in modern office">
```

### Fix Missing Width/Height
```html
<!-- Antes (riesgo CLS) -->

<!-- Después (evita el cambio) -->
```

## Impacto en el rendimiento

La optimización de las imágenes puede mejorar:
- **Core Web Vitals**
  - LCP: entrega de imágenes de héroe más rápida
  - CLS: evita el cambio de diseño de imágenes sin tamaño
- **Page Speed**: Carga entre un 20% y un 40% más rápida con formatos modernos
- **User Experience**: Carga de página más rápida, mejor rendimiento percibido
- **SEO**: Mejor puntuación de Core Web Vitals → posible aumento de clasificación

## Resumen de mejores prácticas

1. **Always include alt text** - Accesibilidad + búsqueda de imágenes.
2. **Always include width/height** - Previene el cambio de diseño (CLS)
3. **Use modern formats** - WebP para un 30% más pequeño, AVIF para un 50%+
4. **Lazy load below-fold** - `loading="lazy"` difiere imágenes no críticas
5. **Prioritize LCP image** - `fetchpriority="high"` en héroe
6. **Make responsive** - `srcset` y `sizes` para tamaños apropiados para el dispositivo
7. **Test before publish** - Utilice PageSpeed ​​Insights para verificar las mejoras
