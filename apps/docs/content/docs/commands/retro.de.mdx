# /retro

Automatisierter Retrospektivgenerator, der Projektdaten analysiert, um Erkenntnisse, Muster und umsetzbare Verbesserungen im Start/Stop/Fortfahren-Format aufzuzeigen.

## Schnellstart

```bash
/agileflow:retro
```

Erstellt anhand der Projektdaten eine Retrospektive der letzten 2 Wochen.

## Parameter

| Parameter | Erforderlich | Standard | Beschreibung |
|-----------|----------|---------|-------------|
| `TIMEFRAME` | Nein | 2 Wochen | Zu analysierender Zeitraum: Sprint, 2 Wochen, 30 Tage oder 90 Tage |
| `EPIC` | Nein | - | Analysieren Sie nur bestimmte Epics (z. B. EP-0010) |
| `FORMAT` | Nein | ASCII | Ausgabeformat: ASCII, Markdown oder HTML |
| `SAVE` | Nein | wahr | Speichern Sie es unter docs/08-project/retrospectives/ |
