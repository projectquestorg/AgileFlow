# /retro

Automatisierter Retrospektivgenerator, der Projektdaten analysiert, um Erkenntnisse, Muster und umsetzbare Verbesserungen im Start/Stop/Fortfahren-Format aufzuzeigen.

## Schnellstart

```bash
/agileflow:retro
```

Erstellt anhand der Projektdaten eine Retrospektive der letzten 2 Wochen.

## Parameter

| Parameter | Required | Default | Description |
|-----------|----------|---------|-------------|
| `TIMEFRAME` | No | 2weeks | Period to analyze: sprint, 2weeks, 30d, or 90d |
| `EPIC` | No | - | Analyze specific epic only (e.g., EP-0010) |
| `FORMAT` | No | ascii | Output format: ascii, markdown, or html |
| `SAVE` | No | true | Save to docs/08-project/retrospectives/ |
