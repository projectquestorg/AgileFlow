# /seo:plan

Generate a strategic SEO plan with industry-specific templates, competitive positioning analysis, and a prioritized improvement roadmap.

## Quick Start

```bash
/agileflow:seo:plan https://example.com                      # 6-month SEO plan (default)
/agileflow:seo:plan https://example.com TIMEFRAME=3mo         # 3-month sprint plan
/agileflow:seo:plan https://example.com TIMEFRAME=12mo        # Annual SEO strategy
```

## Parameters

| Parameter | Required | Default | Description |
|-----------|----------|---------|-------------|
| `<URL>` | Yes | - | Site to plan for |
| `TIMEFRAME` | No | 6mo | Planning horizon: 3mo, 6mo, or 12mo |

## Planning Phases by Timeframe

### 3-Month Plan
- **Month 1**: Critical fixes (blocking issues)
- **Month 2**: Foundation (technical + schema)
- **Month 3**: Content optimization

### 6-Month Plan
- **Month 1-2**: Critical fixes + technical foundation
- **Month 3-4**: Content strategy + schema implementation
- **Month 5-6**: Authority building + performance optimization

### 12-Month Plan
- **Q1**: Foundation (technical + critical fixes)
- **Q2**: Content + schema
- **Q3**: Authority + link building
- **Q4**: Advanced optimization + monitoring

## Business Type Templates

Each business type gets industry-specific recommendations:

### SaaS
**Priority Areas**: Content marketing, technical SEO, schema
- Build blog strategy (industry insights, how-tos, case studies)
- Implement structured data (Organization, Product, FAQPage)
- Optimize documentation pages for search
- Create topic clusters around core features

### Local Business
**Priority Areas**: Local SEO, Google Business, reviews
- Optimize Google Business Profile
- Ensure NAP consistency across web
- Build location pages if multi-location
- Generate and manage reviews

### E-commerce
**Priority Areas**: Product SEO, site structure, Core Web Vitals
- Implement Product schema on all products
- Optimize faceted navigation
- Improve Core Web Vitals (LCP, CLS)
- Create category pages with unique content

### Publisher
**Priority Areas**: Content quality, E-E-A-T, freshness
- Establish author credentials/bios
- Create topic authority with clusters
- Implement Article schema
- Focus on original research/data

### Agency
**Priority Areas**: Authority, portfolio, expertise
- Showcase case studies with metrics
- Build expertise/author pages
- Implement Service schema
- Generate testimonials and social proof

all products, expand category content, and build authority.

Expected impact: 30-50% increase in organic traffic, improved rich
result eligibility.

## Phase 1: Foundation (Months 1-2)

### Goals
- Add Product schema to all 350+ products
- Fix 10 broken URLs in sitemap
- Establish content standards

### Actions

#### Month 1: Schema Implementation
- [ ] Generate Product schema for all products (estimated 40 hours)
  - Include price, availability, reviews
  - Set returnPolicyCountry (required March 2025)
  - Expected impact: Enable product rich results
- [ ] Add Organization schema to homepage
  - Expected impact: Knowledge panel
- [ ] Test all schema in Rich Results Test
  - Expected impact: Verify implementation

#### Month 2: Content Standards
- [ ] Audit category pages for depth
  - Minimum 500 words per category
  - Add competitive differentiators
  - Expected impact: Better category rankings
- [ ] Update stale product descriptions
  - Min 100 words, unique per product
  - Include use cases and benefits
  - Expected impact: Improved CTR and conversion

### KPIs to Track
- Schema coverage: 0% → 100% (products)
- Average product description length: 45w → 120w
- Broken URLs in sitemap: 10 → 0

---

## Phase 2: Content & Authority (Months 3-4)

### Goals
- Build content authority
- Improve E-E-A-T signals
- Expand content beyond product pages

### Actions

#### Month 3: Blog Strategy
- [ ] Create 4 pillar content pieces

  - Include comparisons, expert tips, data
  - Expected impact: Brand authority, organic traffic +15%

#### Month 4: Link Building & Authority
- [ ] Identify 20 relevant industry publications
  - Pitch guest posts or features
  - Expected impact: Backlinks + brand awareness
- [ ] Create expert bios for team members
  - LinkedIn profiles, author pages
  - Expected impact: E-E-A-T + YMYL trust signals

### KPIs to Track

## Phase 3: Optimization (Months 5-6)

### Goals
- Performance optimization

### Actions

  - All product gallery images
  - Expected impact: Faster page load

#### Month 6: Advanced Schema & Measurement

  - Aggregate reviews from customers
  - Enable star rating rich results
  - Expected impact: +5-10% CTR from search
- [ ] Set up reporting dashboard
  - Weekly ranking tracking

## Tips for Success

1. **Start with quick wins** - Schema, broken links, freshness
2. **Content takes time** - Plan content calendar 1-2 months ahead
3. **Link building is slow** - Start pitches early in the plan
4. **Monitor continuously** - Weekly check-ins help catch issues
5. **Be flexible** - Adjust based on what's working
6. **Document learnings** - Share insights across the team

## Budget Considerations

- **DIY (your team)**: 300-400 hours @ $50-100/hr = $15k-40k
- **Agency (full service)**: $5k-20k/month = $30k-120k for 6 months
- **Hybrid**: In-house team + agency for specialist work = $20k-60k
