# /epic:lista

Exiba todos os épicos com status, acompanhamento de progresso e opções de ação rápida.

## Início rápido

```bash
/agileflow:epic:list
```

## Parâmetros

| Parâmetro | Obrigatório | Padrão | Descrição |
|-----------|----------|---------|------------|
| `STATUS` | Não | - | Filtrar por status: `active`, `complete`, `on-hold` |

## Exemplos

### Listar todos os épicos

```bash
/agileflow:epic:list
```

Mostra todos os épicos com status e porcentagem de conclusão da história.

### Listar apenas épicos ativos

```bash
/agileflow:epic:list STATUS=active
```

Filtros para mostrar apenas os épicos em andamento.

### Listar apenas épicos concluídos

Mostra épicos que têm todas as histórias concluídas.

## Definições de status

| Estado | Significado | Barra de progresso | Ação |
|----|---------|------|--------|
| **ativo** | Em andamento, aceitando novas histórias | Preenchimento parcial | Continue o trabalho, adicione histórias |
| **completo** | Todas as histórias concluídas | Completo (100%) | Ver resumo, possivelmente reabrir |
| **em espera** | Pausado, atualmente não trabalhado | Congelado | Retomar ou fechar |

## Visualização do progresso

O progresso é mostrado como:
- **Numerador**: histórias concluídas
- **Denominador**: Total de histórias em épico
- **Porcentagem**: conclusão geral

Exemplo: __NOTRASLATE_19__ = 4 histórias concluídas, 2 restantes, 67% concluídas

## Interpretando resultados

### Alto progresso (acima de 80%)

```
EP-0002 | Authentication | active | 5/6 (83%) | AG-API
```

- Epic está quase completo
- Concentre-se em terminar a última história
- Planeje o próximo épico para começar
- Não feche até 100%

### Progresso Médio (40-80%)

__NÃOTRADUZIR_13__

- Epic está no meio do sprint
- Pode precisar de histórias adicionais
- Verifique se as histórias estão bloqueadas
- Considere adicionar recursos

### Baixo progresso (abaixo de 40%)

__NOTRADUZIR_14__

- Épico acabou de começar
- Aceleração em andamento
- Monitorar bloqueadores
- Pode querer adicionar mais histórias
