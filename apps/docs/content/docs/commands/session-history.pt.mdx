# /sessão:história

Exiba o histórico de todas as sessões, incluindo as encerradas, com métricas de tempo e atividade.

## Início rápido

```bash
/agileflow:session:history
```

## Parâmetros

| Parâmetro | Obrigatório | Padrão | Descrição |
|-----------|----------|---------|------------|
| `DAYS` | Não | `7` | Número de dias de histórico a mostrar |

## Exemplos

```bash
# Last 7 days (default)
/agileflow:session:history

# Last 30 days
/agileflow:session:history DAYS=30

# All history
/agileflow:session:history DAYS=all
```

## Dados históricos

O histórico da sessão inclui:

| Campo | Descrição |
|-------|------------|
| ID da sessão | Identificador numérico |
| Apelido | Nome amigável, se fornecido |
| Caminho | Localização do diretório |
| Filial | Ramo Git usado |
| Estado | ativo, encerrado ou obsoleto |
| Criado | Quando a sessão foi iniciada |
| Terminou | Quando a sessão terminou (se aplicável) |
| Duração | Quanto tempo a sessão esteve ativa |
| Árvore de trabalho | Se a árvore de trabalho foi mantida ou removida |

## Data Storage

Session history is stored in `.agileflow/sessions/registry.json`:

```json
{
  "sessions": {
    "1": {
      "created": "2025-12-20T10:00:00Z",
      "last_active": "2025-12-26T15:00:00Z",
      "is_main": true
    },
    "2": {
      "created": "2025-12-24T14:00:00Z",
      "ended": "2025-12-25T16:30:00Z",
      "nickname": "auth",
      "worktree_removed": true
    }
  }
}
```
