# /sesión:historial

Muestra el historial de todas las sesiones, incluidas las finalizadas, con métricas de tiempo y actividad.

## Inicio rápido

```bash
/agileflow:session:history
```

## Parámetros

| Parámetro | Requerido | Predeterminado | Descripción |
|-----------|----------|---------|-------------|
| `DAYS` | No | `7` | Número de días de historia para mostrar |

## Ejemplos

```bash
# Last 7 days (default)
/agileflow:session:history

# Last 30 days
/agileflow:session:history DAYS=30

# All history
/agileflow:session:history DAYS=all
```


## Datos históricos

El historial de la sesión incluye:

| Campo | Descripción |
|-------|-------------|
| ID de sesión | Identificador numérico |
| Apodo | Nombre amigable para los humanos, si se proporciona |
| Camino | Ubicación del directorio |
| Sucursal | Rama de Git utilizada |
| Estado | activo, finalizado o obsoleto |
| Creado | Cuando se inició la sesión |
| Terminado | Cuándo finalizó la sesión (si corresponde) |
| Duración | Cuánto tiempo estuvo activa la sesión |
| Árbol de trabajo | Si el árbol de trabajo se mantuvo o se eliminó |

## Casos de uso

### Revisar patrones de trabajo

```bash
/agileflow:session:history DAYS=30
# See how many sessions you've used
# Identify if you're creating too many abandoned sessions
```

### Encontrar sesiones antiguas

```bash
/agileflow:session:history DAYS=all
# Find a session from weeks ago
# Check if its worktree still exists
```

### Limpieza Dedecisiones

```bash
/agileflow:session:history
# See which sessions have worktrees kept
# Decide which to clean up
```

## Almacenamiento de datos

El historial de sesiones se almacena en `.agileflow/sessions/registry.json`:

```json
{
  "sessions": {
    "1": {
      "created": "2025-12-20T10:00:00Z",
      "last_active": "2025-12-26T15:00:00Z",
      "is_main": true
    },
    "2": {
      "created": "2025-12-24T14:00:00Z",
      "ended": "2025-12-25T16:30:00Z",
      "nickname": "auth",
      "worktree_removed": true
    }
  }
}
```
