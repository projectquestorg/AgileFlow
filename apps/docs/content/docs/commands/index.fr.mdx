# Commandes

AgileFlow fournit **85 slash commands** que vous pouvez utiliser directement dans Claude Code. Toutes les commandes suivent le modèle `/agileflow:command-name`.

| [`/story`](/commands/story) | Create a user story with acceptance criteria (subcommands: list, view) |

## Modèles d'utilisation

### Utilisation de base

```bash
# Simple command
/agileflow:board

# Command with required parameters
/agileflow:status STORY=US-0001 STATUS=in-progress

# Command with optional parameters
/agileflow:metrics TIMEFRAME=30d EPIC=EP-0001 FORMAT=markdown
```

### Conventions des paramètres

- **Required parameters**: `PARAM=<value>` - Doit être fourni
- **Optional parameters**: `[PARAM=<value>]` - Peut être omis pour les valeurs par défaut
- **IDs**: Utilisez des modèles comme `EP-0001`, `US-0001`, `AG-UI`
- **Lists**: Valeurs séparées par des virgules là où elles sont indiquées

### Obtenir de l'aide

```bash
# Show all available commands
/agileflow:help

# System diagnostics
/agileflow:diagnose
```
