# / Agileflow:context:full

قم بإنشاء أو تحديث ملخص السياق الشامل لأدوات الذكاء الاصطناعي على الويب.

## بداية سريعة

__لا تترجم_0__

## الغرض

إنشاء أو تحديث `docs/context.md` بأقسام مُدارة تحتوي على:

- ما نبنيه والتركيز الحالي
- خريطة الميزات وملخص التكنولوجيا / الأدوات
- القرارات الرئيسية (ADRs) ولمحة سريعة عن الهندسة المعمارية
- حالة الاختبار وCI
- التقدم الأخير (آخر 10 رسائل حافلة)
- المخاطر والخطوات التالية

تم تصميم هذا الملف ليتم لصقه في ChatGPT أو Perplexity أو Gemini أو Claude web لإعطاء سياق المشروع الكامل لأدوات الذكاء الاصطناعي الخارجية.

- **بعد الإنجازات الرئيسية** - التقط العمل المكتمل
- **قبل جلسات الذكاء الاصطناعي على الويب** - تأكد من أن السياق حديث

- **أثناء التخطيط للسباق** - قم بتحديث مجالات التركيز
