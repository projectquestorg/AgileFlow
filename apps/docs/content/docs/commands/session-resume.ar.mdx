# / الجلسة: استئناف

قم بالتبديل إلى جلسة Claude Code مختلفة. يعرض الجلسات المتاحة ويوفر الأمر للتبديل.

## بداية سريعة

```bash
/agileflow:session:resume
```

## كيف يعمل

1. يسرد كافة الجلسات المسجلة مع الحالة
2. يقدم منتقي تفاعلي لتحديد الجلسة
3. يظهر الأمر الطرفي للتبديل

## مثال التفاعل

```
Which session do you want to resume?

○ Session 1 (main) - /home/user/myproject [active]
○ Session 2 "auth" - /home/user/myproject-auth [active]
○ Session 3 - /home/user/myproject-3 [stale]
```

بعد اختيار الجلسة الثانية:

```
To resume Session 2 "auth":

  cd /home/user/myproject-auth && claude --resume

Note: Use a new terminal window/tab for best experience.
```

## دول الجلسة

| State | Can Resume? | Notes |
|-------|-------------|-------|
| `active` | Yes | Another Claude instance is running there |
| `stale` | Yes | Previous session ended, directory still exists |
| `current` | No | You're already in this session |

## العلم --resume

ال `claude --resume` يخبر العلم كلود كود بما يلي:
- قم بتحميل سجل المحادثة من الجلسة السابقة
- قم باستعادة السياق حول ما كنت تعمل عليه
- استمر من حيث توقفت

بدون `--resume`يبدأ كلود بسياق جديد.

## أمثلة سير العمل

### التبديل بين المهام

```bash
# Currently in Session 1 working on feature A
/agileflow:session:resume
# Select Session 2 "auth"

# In new terminal:
cd /home/user/myproject-auth && claude --resume
# Continue auth work with full context
```

### العودة بعد الاستراحة

```bash
# Start Claude Code
claude

# See: "2 other active sessions"
/agileflow:session:resume
# Pick the session you want to continue
```

### التنظيف قبل الاستئناف

إذا ظهرت الجلسة على أنها قديمة:

```bash
/agileflow:session:status
# Session 3 shows "stale"

/agileflow:session:end
# Select Session 3 to clean up

/agileflow:session:resume
# Now switch to a clean session
```

## نصائح المحطة

**Best practice**: افتح علامة تبويب/نافذة طرفية جديدة لكل جلسة.

```bash
# Tab 1: Session 1 (main project)
cd /home/user/myproject && claude

# Tab 2: Session 2 (auth feature)
cd /home/user/myproject-auth && claude --resume

# Tab 3: Session 3 (API work)
cd /home/user/myproject-3 && claude --resume
```
