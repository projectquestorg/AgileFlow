# / الجلسة: استئناف

قم بالتبديل إلى جلسة Claude Code مختلفة. يعرض الجلسات المتاحة ويوفر الأمر للتبديل.

## بداية سريعة

__لا تترجم_0__

## كيف يعمل

1. يسرد كافة الجلسات المسجلة مع الحالة
2. يقدم منتقي تفاعلي لتحديد الجلسة
3. يظهر الأمر الطرفي للتبديل

## مثال للتفاعل

__لا تترجم_1__

بعد اختيار الجلسة الثانية:

__لا تترجم_2__

## حالات الجلسة

| الدولة | يمكن استئناف؟ | ملاحظات |
|-------|------------|-------|
| __غير مترجم_7__ | نعم | يتم تشغيل مثيل آخر لكلود هناك |
| __غير مترجم_8__ | نعم | انتهت الجلسة السابقة، ولا يزال الدليل موجودًا |
| __غير مترجم_9__ | لا | أنت بالفعل في هذه الجلسة |

## علامة --resume

تخبر العلامة `claude --resume` كلود كود بما يلي:
- تحميل سجل المحادثات من الجلسة السابقة
- استعادة السياق حول ما كنت تعمل عليه
- تابع من حيث توقفت

بدون `--resume`، يبدأ كلود بسياق جديد.
