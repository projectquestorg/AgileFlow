# /configurar

Configure recursos avançados do AgileFlow com perfis, opções de ativação/desativação e migração de formato para personalizar a configuração do seu fluxo de trabalho.

## Início rápido

```bash
/agileflow:configure
```

Isso abre um menu de configuração interativo. Você também pode executar com opções específicas:

```bash
/agileflow:configure --profile=full
/agileflow:configure --enable=sessionstart,precompact
/agileflow:configure --disable=archival
```

## Parâmetros

| Parâmetro | Obrigatório | Padrão | Descrição |
|-----------|----------|---------|------------|
| `--profile` | Não | - | Perfil de configuração: `full`, `basic`, `minimal` ou `none` |
| `--enable` | Não | - | Lista separada por vírgulas de recursos a serem ativados |
| `--disable` | Não | - | Lista separada por vírgulas de recursos a serem desativados |
| `--migrate` | Não | - | Corrigir problemas de formato na configuração existente |
| `--detect` | Não | - | Verifique o status da configuração atual |
| `--archival-days` | Não | 7 | Dias antes de arquivar histórias concluídas |

## Perfis de configuração

| Perfil | Início da sessão | Pré-Compacto | Pare o Gancho | Arcohival | Linha de status |
|---------|---------|------------|-----------|----------|------------|
| `full` | ✅ | ✅ | ✅ | ✅ 7 dias | ✅ |
| `basic` | ✅ | ✅ | ❌ | ✅ 7 dias | ❌ |
| `minimal` | ✅ | ❌ | ❌ | ✅ 7 dias | ❌ |
| `none` | ❌ | ❌ | ❌ | ❌ | ❌ |

## Recursos disponíveis

- **sessionstart**: exibição de boas-vindas com status do projeto no início da sessão
- **pré-compactado**: preserva o contexto ao compactar conversas
- **stop**: avisa sobre alterações não confirmadas do git antes de encerrar a sessão
- **arquivamento**: arquiva automaticamente histórias antigas concluídas (dias configuráveis)
- **linha de status**: barra de status personalizada mostrando a história atual e o progresso

## Exemplos

### Configuração básica (recomendado)

```bash
/agileflow:configure --profile=full
```

Ativa todos os recursos: ganchos, arquivamento e linha de status.

### Habilitar recursos específicos

### Desativar recurso

### Corrigir problemas de configuração

__NOTRADUZIR_5__

Se você vir avisos de formato, isso irá corrigi-los automaticamente e criar um backup em __NOTRASLATE_23__.

### Verifique o status atual

__NOTRADUZIR_6__

Mostra o que está ativado ou desativado no momento sem fazer alterações.

### Limite de arquivamento personalizado

__NÃOTRADUZIR_7__

Arquive histórias concluídas após 14 dias, em vez dos 7 dias padrão.

## O que é configurado

### Sistema de Ganchos
Configura ganchos em __NOTRASLATE_24__ para:
- SessionStart: exibe informações do projeto quando você inicia uma sessão
- UserPromptSubmit: executa a validação antes dos commits
- Parar: avisa sobre alterações não confirmadas ao encerrar a sessão

### Sistema de Arquivo
Atualiza __NOTRASLATE_25__ para permitir o arquivamento automático de histórias concluídas anteriores ao limite especificado.

### Linha de status
Configura a barra de status personalizada do Claude Code mostrando:
- Ramo git atual
- ID da história ativa
- Contagem de WIP (trabalho em andamento)
- Porcentagem de conclusão

## Notas importantes

- **Migração de formato**: se você vir avisos de `⚠️ INVALID FORMAT`, execute `--migrate` primeiro, antes de outras opções
- **Backup Criado**: A opção de migração cria automaticamente um backup em __NOTRASLATE_28__
- **É necessário reiniciar**: você deve sair completamente e reiniciar o Claude Code para que as alterações sejam aplicadas
- **Metadados atualizados**: a configuração do recurso é rastreada em `docs/00-meta/agileflow-metadata.json`
