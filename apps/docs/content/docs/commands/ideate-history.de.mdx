# /ideate:Geschichte

Fragen Sie frühere Ideenberichte ab, verfolgen Sie den Ideenstatus, vergleichen Sie Berichte im Zeitverlauf und identifizieren Sie wiederkehrende Themen.

## Schnellstart

```bash
/agileflow:ideate:history
```

## Argumente

| Argument | Description |
|----------|-------------|
| `[IDEA-XXXX]` | View specific idea by ID |
| `STATUS=pending\|in-progress\|implemented\|recurring` | Filter by status |
| `MODE=trends` | Show aggregate statistics across all reports |
| `COMPARE=report1,report2` | Diff two reports |

## Merkmale

- **Status filtering** - Finden Sie Ideen anhand ihres aktuellen Status
- **Recurring detection** - Hebt Ideen hervor, die in mehr als 2 Berichten erschienen sind
- **Trend analysis** - Aggregieren Sie Statistiken über alle Ideenberichte
- **Report comparison** - Unterschied zwischen zwei Berichten, die „Gelöst/Neu/Persistiert/Verworfen“ anzeigen

## Beispiele

```bash
# View all pending ideas
/agileflow:ideate:history STATUS=pending

# Check trends across reports
/agileflow:ideate:history MODE=trends

# Compare two reports
/agileflow:ideate:history COMPARE=ideation-20260101.md,ideation-20260201.md
```
