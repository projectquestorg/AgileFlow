# /session:historique

Affichez l'historique de toutes les sessions, y compris celles terminées, avec des mesures de timing et d'activité.

## Démarrage rapide

```bash
/agileflow:session:history
```

## Paramètres

| Paramètre | Obligatoire | Par défaut | Descriptif |
|-----------|----------|---------|-------------|
| `DAYS` | Non | `7` | Nombre de jours d'historique à afficher |

## Exemples

```bash
# Last 7 days (default)
/agileflow:session:history

# Last 30 days
/agileflow:session:history DAYS=30

# All history
/agileflow:session:history DAYS=all
```

## Données d'historique

L'historique de la session comprend :

| Champ | Descriptif |
|-------|-------------|
| ID de session | Identifiant numérique |
| Surnom | Nom convivial si fourni |
| Chemin | Emplacement du répertoire |
| Branche | Branche Git utilisée |
| Statut | actif, terminé ou obsolète |
| Créé | Quand la session a démarré |
| Terminé | Quand la session s'est terminée (le cas échéant) |
| Durée | Combien de temps la session a été active |
| Arbre de travail | Si l'arbre de travail a été conservé ou supprimé |

## Cas d'utilisation

### Révision des modèles de travail

```bash
/agileflow:session:history DAYS=30
# See how many sessions you've used
# Identify if you're creating too many abandoned sessions
```

### Recherche d'anciennes sessions

```bash
/agileflow:session:history DAYS=all
# Find a session from weeks ago
# Check if its worktree still exists
```

### Nettoyage Dedécisions

```bash
/agileflow:session:history
# See which sessions have worktrees kept
# Decide which to clean up
```

## Stockage des données

L'historique de la session est stocké dans `.agileflow/sessions/registry.json` :

```json
{
  "sessions": {
    "1": {
      "created": "2025-12-20T10:00:00Z",
      "last_active": "2025-12-26T15:00:00Z",
      "is_main": true
    },
    "2": {
      "created": "2025-12-24T14:00:00Z",
      "ended": "2025-12-25T16:30:00Z",
      "nickname": "auth",
      "worktree_removed": true
    }
  }
}
```
