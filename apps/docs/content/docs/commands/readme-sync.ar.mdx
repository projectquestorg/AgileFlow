## كيف يعمل

### الخطوة 1: قائمة المحتويات
يقوم الأمر بفحص المجلد بحثًا عن:
- كافة تخفيض السعر وغيرها من الملفات
- الدلائل الفرعية ومحتوياتها

### الخطوة 2: استخراج الأوصاف
لكل ملف، يستخرج:
- **ملفات تخفيض السعر**: العنوان الأول (# Heading) أو الجملة الأولى
- **ملفات أخرى**: السطر الأول أو الوصف المستند إلى اسم الملف

### الخطوة 3: إنشاء قسم المحتويات
يُنشئ تنسيق قائمة نقطية نظيفًا باستخدام:
- أسماء الملفات بالخط العريض
- أوصاف جملة واحدة
- بنية متداخلة للأدلة الفرعية

### الخطوة 4: المعاينة والتأكيد

## ما يتم تحديثه

يقوم الأمر ** بتحديث القسم `## Contents` فقط**:

- يزيل قوائم الملفات القديمة
- يضيف كافة الملفات الحالية مع الأوصاف
- يحافظ على جميع أقسام README الأخرى دون تغيير
- يحافظ على الملاحظات والروابط المخصصة
- يحافظ على أقسام الرأس والمقدمة والتذييل سليمة

وهذا يعني أنه يمكنك استخدام الأمر بأمان دون فقدان الوثائق المخصصة.
