# Comandos

AgileFlow proporciona **74 comandos de barra diagonal** que puede usar directamente en Claude Code. Todos los comandos siguen el patrón `/agileflow:command-name`.

|---------|-------------|

## Patrones de uso

### Uso básico

```bash
# Simple command
/agileflow:board

# Command with required parameters
/agileflow:status STORY=US-0001 STATUS=in-progress

# Command with optional parameters
/agileflow:metrics TIMEFRAME=30d EPIC=EP-0001 FORMAT=markdown
```

### Convenciones de parámetros

- **Required parameters**: `PARAM=<value>` - Must be provided
- **Parámetros opcionales**: `[PARAM=<value>]` - Se puede omitir por defecto
- **ID**: use patrones como `EP-0001`, `US-0001`, `AG-UI`
- **Listas**: valores separados por comas donde se indique

### Obteniendo ayuda

```bash
# Show all available commands
/agileflow:help

# System diagnostics
/agileflow:diagnose
```
