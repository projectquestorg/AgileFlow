# / سيو: الصور

تحليل تحسين الصورة على الصفحة: جودة النص البديل، والتحجيم الواضح لمنع CLS، والتنسيقات الحديثة (WebP/AVIF)، والتحميل البطيء، وأولوية صورة LCP، والصور سريعة الاستجابة.

## بداية سريعة

```bash
/agileflow:seo:images https://example.com                    # Image audit for homepage
/agileflow:seo:images https://example.com/products/item       # Product page images
```

## حدود

| Parameter | Required | Description |
|-----------|----------|-------------|
| `<URL>` | Yes | Page to analyze |

## ما يتم فحصه

| Aspect | Weight | Impact |
|--------|--------|--------|
| Alt text quality | 30% | Accessibility + image search rankings |
| Image sizing (CLS) | 20% | Core Web Vitals - layout stability |
| Modern formats | 15% | Page speed + LCP |
| Lazy loading | 15% | Page speed |
| LCP optimization | 10% | Core Web Vitals - largest paint |
| Responsive images | 10% | Mobile performance |

## نص بديل التهديف

| Quality | Criteria | Score Impact |
|---------|----------|------------|
| **Good** | Descriptive, 10-125 chars, contextual keyword | +points |
| **Acceptable** | Present but generic ("image", "photo") | Neutral |
| **Poor** | Filename-based ("IMG_001.jpg", "screenshot") | -2 points |
| **Missing** | No alt attribute at all | -3 points (HIGH severity) |
| **Decorative** | Empty alt="" on decorative images | Correct (no deduction) |
| **Keyword-stuffed** | Excessive keywords crammed in | -2 points |

## إرشادات النص البديل

نص بديل جيد:
- يصف ما تظهره الصورة
- يوصى باستخدام 10-125 حرفًا
- يتضمن الكلمات الرئيسية ذات الصلة بشكل طبيعي
- لا يقول "صورة" أو "صورة"
- للصور الزخرفية، استخدم `alt=""`

مثال لنص بديل جيد:
- "امرأة محترفة تكتب على الكمبيوتر المحمول في مكتب المكتب الحديث"
- "لقطة شاشة للوحة تحكم Google Search Console تعرض مقاييس حركة المرور"
- "صورة المنتج: سماعات رأس لاسلكية زرقاء مع خاصية إلغاء الضوضاء"

مثال على نص بديل رديء:
- "IMG_001.jpg" ❌
- "الصورة" ❌
- "صورة الأشياء" ❌
- "سماعات الرأس الزرقاء، اللاسلكية، إلغاء الضوضاء، اشتري الآن" (حشو الكلمات الرئيسية) ❌

## توصيات تنسيق الصورة

| Format | Use When | Not When |
|--------|----------|----------|
| WebP | Modern format, wide support | Supporting very old browsers |
| AVIF | Best compression, growing support | Older browser support needed |
| JPEG | Photos, realistic images | Transparency needed |
| PNG | Transparency needed | Photos (use JPEG) |
| GIF | Animations | Use MP4 video instead |
| SVG | Icons, illustrations, logos | Photos |
| BMP/TIFF | Never | Never |

## صور مستجيبة

للتسليم الأمثل على الأجهزة المحمولة وسطح المكتب:

```html
<!-- Good: Responsive with srcset and sizes -->
<img
  src="image-800.jpg"
  srcset="image-400.jpg 400w, image-800.jpg 800w, image-1200.jpg 1200w"
  sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px"
  width="800"
  height="600"
  alt="Descriptive alt text"
  loading="lazy"
>

<!-- Better: Format alternatives with <picture> -->
<picture>
  <source type="image/avif" srcset="image.avif">
  <source type="image/webp" srcset="image.webp">
  <img src="image.jpg" alt="Descriptive alt text" width="800" height="600">
</picture>
```

## تحسين صورة LCP

لبطلك/الصورة الأكبر:

```html
<!-- Good: LCP image gets priority -->
<img
  src="hero.jpg"
  alt="Hero section describing main benefit"
  width="1200"
  height="600"
  fetchpriority="high"
  loading="eager"
/>

<!-- With preload for extra speed -->
<link rel="preload" as="image" href="hero.jpg">
<img src="hero.jpg" alt="..." width="1200" height="600">
```

## تأثير الأداء

يمكن أن يؤدي تحسين الصور إلى تحسين:
- **Core Web Vitals**
  - LCP: تسليم أسرع لصور البطل
  - CLS: يمنع تغيير التخطيط من الصور غير الحجم
- **Page Speed**: تحميل أسرع بنسبة 20-40% مع التنسيقات الحديثة
- **User Experience**: تحميل أسرع للصفحة، وأداء أفضل
- **SEO**: نتيجة أفضل لمؤشرات أداء الويب الأساسية → تعزيز محتمل للتصنيف

## ملخص أفضل الممارسات

1. **Always include alt text** - إمكانية الوصول + البحث عن الصور
2. **Always include width/height** - يمنع تغيير التخطيط (CLS)
3. **Use modern formats** - WebP أصغر بنسبة 30%، وAVIF بنسبة 50%+
4. **Lazy load below-fold** - `loading="lazy"` يؤجل الصور غير الحرجة
5. **Prioritize LCP image** - `fetchpriority="high"` على البطل
6. **Make responsive** - `srcset` و `sizes` للأحجام المناسبة للجهاز
7. **Test before publish** - استخدم PageSpeed ​​Insights للتحقق من التحسينات
