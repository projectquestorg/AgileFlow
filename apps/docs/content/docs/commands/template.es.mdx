## Sintaxis de plantilla

### Formato variable

Las plantillas utilizan llaves dobles para las variables:

```markdown
# {{TITLE}}

**Created by**: {{AUTHOR}}
**Date**: {{DATE}}
**Status**: {{STATUS}}
```

### Variables incorporadas

Disponible en todas las plantillas:

| Variable | Example | Description |
|----------|---------|-------------|
| `{{DATE}}` | 2025-12-24 | Current date (YYYY-MM-DD) |
| `{{DATETIME}}` | 2025-12-24T10:30:00Z | ISO 8601 datetime |
| `{{USER}}` | alice | Current git user |
| `{{YEAR}}` | 2025 | Current year |

### Variables de plantilla de historia

Para plantillas espec铆ficas de historias:

```text
{{STORY_ID}} - Story identifier (US-0042)
{{EPIC_ID}} - Parent epic (EP-0001)
{{OWNER}} - Story owner/assignee
{{ESTIMATE}} - Story point estimate
{{AC}} - Acceptance criteria
{{DEPENDENCIES}} - Dependencies on other stories
```

### Variables de plantilla ADR

Para registros de decisiones de arquitectura:

```text
{{NUMBER}} - ADR number (0001, 0042)
{{STATUS}} - Decision status (proposed, accepted, superseded)
{{CONTEXT}} - Driving factors
{{DECISION}} - The decision made
{{CONSEQUENCES}} - Trade-offs and impacts
```
