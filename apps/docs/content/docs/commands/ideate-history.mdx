# /ideate:history

Query past ideation reports, track idea statuses, compare reports over time, and identify recurring themes.

## Quick Start

```bash
/agileflow:ideate:history
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[IDEA-XXXX]` | View specific idea by ID |
| `STATUS=pending\|in-progress\|implemented\|recurring` | Filter by status |
| `MODE=trends` | Show aggregate statistics across all reports |
| `COMPARE=report1,report2` | Diff two reports |

## Features

- **Status filtering** - Find ideas by their current status
- **Recurring detection** - Highlights ideas that appeared in 2+ reports
- **Trend analysis** - Aggregate statistics across all ideation reports
- **Report comparison** - Diff two reports showing Resolved/New/Persisted/Dropped

## Examples

```bash
# View all pending ideas
/agileflow:ideate:history STATUS=pending

# Check trends across reports
/agileflow:ideate:history MODE=trends

# Compare two reports
/agileflow:ideate:history COMPARE=ideation-20260101.md,ideation-20260201.md
```
