# Recursos

AgileFlow inclui recursos poderosos para proteger sua base de código, automatizar fluxos de trabalho e manter sua equipe produtiva.

## Recursos principais

### Sistema de habilidades

Crie, gerencie e melhore continuamente prompts de IA personalizados que aprendem com seu feedback.

- **Criação apoiada por pesquisa** - Gerado a partir de documentos da web, documentação local e sua base de código
- **Fluxos de trabalho do livro de receitas** - Vários guias específicos de casos de uso em uma única habilidade
- **Autoaperfeiçoamento** - As habilidades são aprendidas com suas correções e refinadas automaticamente
- **Integração MCP** - Conecte-se aos servidores Model Context Protocol para acesso à ferramenta
- **Compartilhável pela equipe** - Comprometa-se com o git e compartilhe conhecimento institucional

[Saiba mais sobre habilidades__NOTRASLATE_11__

### Sistema de experiência de agente

Habilite agentes de autoaperfeiçoamento que mantêm o conhecimento do domínio e aceleram a execução de tarefas.

- **Arquivos de expertise** - Os agentes mantêm modelos mentais de seu domínio
- **Fluxo de trabalho em três etapas** - Planejar (com experiência) → Construir → Melhorar automaticamente
- **Aprendizagem automática** - Eatualizações de experiência à medida que os agentes trabalham e fazem alterações
- **Validação e métricas** - Ferramentas para garantir que o conhecimento permaneça preciso e atualizado
- **Domínios focados** - Conhecimento especializado em banco de dados, API, UI, testes, CI/CD e muito mais

[Saiba mais sobre a experiência do agente__NOTRASLATE_12__

### Sessões Paralelas

Trabalhe em vários recursos, bugs ou experimentos simultaneamente com espaços de trabalho isolados e proteção de limites.

- **Árvores de trabalho Git** - Cada sessão recebe seu próprio diretório e branch
- **Proteção de limite** - Ganchos de edição/gravação evitam edições acidentais entre sessões
- **Troca fácil** - Use __NOTRASLATE_0__ para trocar de sessão no mesmo terminal
- **Acompanhamento de sessão** - AgileFlow rastreia todas as sessões com status e atividade
- **Contexto independente** - Cada sessão mantém sua própria história e progresso

## Automação

### Modo Loop

Execute Claude de forma autônoma com execução contínua de tarefas até a conclusão.

- **Execução autônoma** - Claude trabalha através de tarefas sem interrupção
- **Limites configuráveis** - Defina limites e tempos limite de iteração
- **Acompanhamento do progresso** - Veja o que está acontecendo em cada etapa
- **Backups seguros** - Verificações de segurança integradas evitam loops descontrolados

[Saiba mais sobre o modo Loop__NOTRASLATE_14__

### Modo Visual

Deixe Claude ver e interagir com interfaces visuais por meio de capturas de tela e automação do navegador.

- **Análise de captura de tela** - Claude pode analisar o conteúdo visual
- **Automação do navegador** - Integração do Playwright para testes na web
- **Verificação da IU** - Valide alterações visuais automaticamente
- **Plataforma cruzada** - Funciona em qualquer sistema operacional com suporte a navegador

[Saiba mais sobre o Modo Visual__NOTRASLATE_15__

### Preservação compacta de contexto

Preserve o conhecimento e o estado críticos do agente durante a compactação automática do contexto.

- **Compactação baseada no contexto** - Regras críticas preservadas quando Claude Code compacta o contexto
- **Preservação baseada em prioridades** - Agentes críticos sempre preservados
- **Continuidade do estado** - Sto atualry, bloqueadores e progresso mantidos
- **Suporte para longas conversas** - Foco sustentado do agente durante dias de trabalho
- **Zero sobrecarga** - Automático, sem necessidade de configuração manual

## Configuração e segurança

### Sistema de configuração

9 agentes especializados para configurar e personalizar seu fluxo de trabalho de desenvolvimento.

- **Configuração do Git** - Inicializar repositório e conexões remotas
- **Sistema Hooks** - Automação orientada a eventos (SessionStart, PreCompact, Stop)
- **Linha de status** - Barra de status personalizada no Claude Code
- **Arquivamento** - Gerenciar automaticamente o tamanho do arquivo status.json
- **CI/CD** - Testes automatizados com GitHub Actions, GitLab, CircleCI
- **Atribuição** - Controle a divulgação da IA em commits do git
- **E mais** - Contexto pré-compacto, verificação, gerenciamento de token

[Saiba mais sobre configuração__NOTRASLATE_17__

### Integrações IDE

Instale e configure o AgileFlow para seu IDE favorito com instalação automática de comandos, agentes e habilidades.

- **Vários IDEs** - Claude Código, Cursor, Windsurf, OpenAI Codex CLI
- **Detecção automática** - Detecte quais IDEs estão configurados em seu projeto
- **Controle de Danos** - Sistema de segurança opcional para Claude Code
- **Conteúdo dinâmico** - Comandos e agentes injetados automaticamente na instalação
- **Suporte a vários IDEs** - Use o mesmo projeto em vários IDEs

[Saiba mais sobre integrações IDE__NOTRASLATE_18__

### Controle de Danos

Proteja sua base de código de comandos destrutivos usando ganchos PreToolUse.

- **Validação do comando Bash** - Bloqueie padrões perigosos como `rm -rf /`, `git push --force`
- **Controle de acesso a arquivos** - Proteja segredos, chaves e arquivos críticos
- **Regras personalizáveis** - Adicione padrões de proteção específicos do projeto
- **Modo perguntar primeiro** - Exigir confirmação para operações arriscadas
- **Três camadas de proteção** - Validação das ferramentas Bash, Edit e Write

[Aprenda sobre Controle de Danos__NOTRASLATE_19__
