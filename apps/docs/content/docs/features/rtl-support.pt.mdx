# Suporte ao idioma RTL

A documentação do AgileFlow oferece suporte a idiomas da direita para a esquerda (RTL), incluindo árabe, hebraico, persa e urdu. Este guia aborda como usar o suporte RTL e como contribuir com traduções.

## Usando o modo RTL

### Alternador de idioma

Clique no ícone de idioma/globo no cabeçalho do site para alternar entre os modos LTR e RTL. O site ajustará automaticamente:

- Direção do texto
- Espelhamento de layout
- Alinhamento de navegação
- Posicionamento de componentes

### Detecção Automática

Se o seu navegador estiver configurado para um idioma RTL, o site iniciará automaticamente no modo RTL. Idiomas RTL suportados:

- Árabe (ar)
- Hebraico (ele)
- Persa/Farsi (fa)
- Urdu (você)

## CSS compatível com RTL

O site usa Tailwind CSS com variantes RTL. Use o prefixo __NOTRASLATE_9__ para estilos específicos de RTL:

```css
/* Example RTL styles */
.my-element {
  margin-left: 1rem;   /* LTR default */
}

[dir="rtl"] .my-element {
  margin-left: 0;
  margin-right: 1rem;  /* RTL override */
}

/* Using Tailwind rtl: variant */
<div className="ml-4 rtl:ml-0 rtl:mr-4">
  Content
</div>
```

### Propriedades Lógicas

Na maioria dos casos, prefira propriedades lógicas CSS que controlam automaticamente a direção:

```css
/* Instead of this */
margin-left: 1rem;
padding-right: 2rem;
text-align: left;

/* Use this */
margin-inline-start: 1rem;
padding-inline-end: 2rem;
text-align: start;
```

Equivalentes do vento favorável:
- `ms-*` = margem-inline-start
-`me-*` = margem-inline-end
- `ps-*` = preenchimento-inline-start
- `pe-*` = preenchimento-inline-end
- __NOTRASLATE_14__ = inserção-inline-start
- `end-*` = inserção-inline-end

### Diretrizes de Tradução

1. **Preservar a formatação**: Mantenha blocos de código, links e componentes intactos
2. **Traduzir texto alternativo**: os atributos alternativos da imagem devem ser traduzidos
3. **Mantenha URLs**: não traduza slugs de URL, a menos que seja absolutamente necessário
4. **Revise os termos técnicos**: alguns termos técnicos podem permanecer em inglês

## Testando layout RTL

1. Alterne para o modo RTL usando o alternador de idioma
2. Verifique se essas áreas estão corretasespelhamento:
   - Barra lateral de navegação (deve estar à direita)
   - Alinhamento do ícone de pesquisa
   - Direção de rolagem do bloco de código
   - Barras de progresso
   - Ícones (alguns devem virar, outros não)

### Ícones que NÃO devem virar

- Ícone de pesquisa
- Ícone de link externo
- Marca de seleção
- Mais/menos
- Fechar (X)

### Ícones que DEVEM virar

- Seta para esquerda/direita
- Chevron esquerda/direita
- Recuo/recuo
- Responder/encaminhar
