# Instalação

AgileFlow é distribuído como um pacote npm que pode ser instalado diretamente no seu projeto, adicionando comandos de barra, agentes e estrutura do projeto.

## Pré-requisitos

- **Node.js** 18.0 ou superior
- **Código Claude**, **Cursor**, **Windsurf** ou **OpenAI Codex CLI**
- Um repositório Git para o seu projeto

## Instalação rápida

```bash
npx agileflow@latest setup
```

É isso! O comando __NOTRASLATE_10__ sempre busca a versão mais recente, então você nunca precisa se preocupar com instalações desatualizadas.

### Atualizações

```bash
npx agileflow@latest update
```

## Após a configuração

```bash
/agileflow:help              # View all commands
/agileflow:configure         # Configure hooks, status line, etc.
/agileflow:babysit           # Interactive mentor for implementation
```

### Comandos Instalados

Todos comos mandos ficam disponíveis como __NOTRASLATE_11__ no Código Claude:

| Comando | Descrição |
|--------|-------------|
| `/agileflow:epic` | Crie um novo épico |
| `/agileflow:story` | Crie uma história de usuário |
| `/agileflow:status` | Atualizar status da história |
| `/agileflow:board` | Ver quadro Kanban |
| `/agileflow:babysit` | Iniciar implementação guiada |

Consulte a [referência completa do comando__NOTRASLATE_44__ para todos os 72 comandos.

## Suporte IDE

AgileFlow funciona com vários IDEs com tecnologia de IA:

| IDE | Estado | Localização de configuração |
|-----|--------|-----------------|
| Código Cláudio | Suportado | `.claude/commands/agileflow/` |
| Cursor | Suportado | `.cursor/commands/agileflow/` |
| Windsurf | Suportado | `.windsurf/workflows/agileflow/` |
| CLI do OpenAI Codex | Suportado | `.codex/skills/` e `~/.codex/prompts/` |
