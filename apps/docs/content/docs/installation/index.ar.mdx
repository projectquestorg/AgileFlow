## المتطلبات الأساسية

- **Node.js** 18.0 أو أعلى
- **Claude Code**، **Cursor**، **Windsurf**، أو **OpenAI Codex CLI**
- مستودع Git لمشروعك

## الخطوات التالية

الآن بعد تثبيت AgileFlow:

1. **قم بإنشاء ملحمتك الأولى**: `/agileflow:epic EPIC=EP-0001 TITLE="My Feature"`
2. **تقسيمها إلى قصص**: `/agileflow:story EPIC=EP-0001 STORY=US-0001 TITLE="First task"`
3. **بدء العمل**: __لارانسلات_24__

استخدم `/agileflow:help` لرؤية جميع الأوامر المتاحة.
