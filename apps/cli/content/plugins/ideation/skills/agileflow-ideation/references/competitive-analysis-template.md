## Win/Loss Signals to Track

| Signal                         | Where to find it                             |
| ------------------------------ | -------------------------------------------- |
| Why customers chose us         | Sales call notes, onboarding surveys         |
| Why customers chose competitor | Lost deal notes, churned user interviews     |
| Public reviews                 | G2, Capterra, Trustpilot, App Store          |
| Feature requests               | Support tickets, forum posts, Twitter/Reddit |
| Pricing sensitivity            | Sales objections, deal slippage patterns     |
| Integration gaps               | Support requests for missing integrations    |

## Competitive Response Framework

| Scenario                          | Response                                                                                |
| --------------------------------- | --------------------------------------------------------------------------------------- |
| Competitor adds a feature we lack | Assess: parity vs. differentiation. Add if table stakes, ignore if not our ICP.         |
| Competitor cuts price             | Don't race to bottom. Compete on value. Quantify ROI difference.                        |
| New competitor enters market      | Document, monitor, don't panic. Validate with customers if it changes buying behavior.  |
| Competitor gets acquired          | Research acquirer's intent. Could mean faster growth (threat) or neglect (opportunity). |
