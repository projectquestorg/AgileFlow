# Ad Copy Formula Guide

**Load this when:** Writing, reviewing, or improving ad copy across any platform or format.

## Core Copywriting Formulas

### AIDA — Attention, Interest, Desire, Action

| Stage     | Goal            | Trigger words                                    |
| --------- | --------------- | ------------------------------------------------ |
| Attention | Stop the scroll | "Stop", "Finally", "Warning", numbers, questions |
| Interest  | Build relevance | "If you...", "For [persona]...", "Imagine..."    |
| Desire    | Create want     | Benefits > features, social proof, scarcity      |
| Action    | Drive click     | "Get", "Start", "Try free", "Claim", "Download"  |

**Template:**

```
[Attention hook — 5 words max]
[Interest — 1 sentence connecting to pain]
[Desire — 1-2 key benefits or social proof]
[Action — clear CTA with value]
```

---

### PAS — Problem, Agitate, Solution

Best for: Search ads, email subject lines, long-form display

| Stage    | Job                 | Example                                          |
| -------- | ------------------- | ------------------------------------------------ |
| Problem  | Name the pain       | "Tired of slow load times?"                      |
| Agitate  | Amplify consequence | "Every second costs conversions — and revenue."  |
| Solution | Position your offer | "AgileFlow cuts deploy time by 60%. Start free." |

**Template:**

```
Headline: [Problem as question or statement]
Description: [Agitate] [Solution + CTA]
```

---

### FAB — Feature, Advantage, Benefit

Convert feature-speak to benefit-speak:

| Feature             | Advantage        | Benefit                       |
| ------------------- | ---------------- | ----------------------------- |
| 99.9% uptime SLA    | Always available | Never lose a sale to downtime |
| AI-powered matching | Faster results   | Find what you need in seconds |
| SOC 2 certified     | Audited security | Your data stays yours         |

---

### BAB — Before, After, Bridge

Best for: Video scripts, testimonial ads, awareness campaigns

```
Before: [Current painful state]
After:  [Desired outcome state]
Bridge: [Your product = the path between]
```

## Headline Formulas by Intent

| Intent                 | Formula                       | Example                                              |
| ---------------------- | ----------------------------- | ---------------------------------------------------- |
| Search (informational) | [Keyword] — [Differentiator]  | "Project Management — Try Free 30 Days"              |
| Search (transactional) | [Verb] [Outcome] [Qualifier]  | "Get More Leads Without Increasing Budget"           |
| Display (awareness)    | [Number] + [Noun] + [Benefit] | "5 Ways Teams Ship 2x Faster"                        |
| Social (disruptive)    | Question or bold claim        | "Your CI/CD pipeline is slower than it needs to be." |
| Retargeting            | Continuity + urgency          | "Still thinking it over? Offer ends Friday."         |

---

## Description Line Templates

**Value prop + CTA:**

> [Primary benefit]. [Secondary benefit or proof]. [CTA with qualifier].

**Social proof + CTA:**

> Trusted by [N] [persona type]. [Key differentiator]. [CTA].

**Objection handling:**

> No [common objection]. [Benefit]. [CTA].
> "No contracts. Cancel anytime. Start your free trial today."

## Power Words by Goal

| Goal        | Words                                           |
| ----------- | ----------------------------------------------- |
| Trust       | Proven, Certified, Guaranteed, Verified, Secure |
| Urgency     | Now, Today, Limited, Expires, Last chance       |
| Exclusivity | Exclusive, Members only, Invite, VIP            |
| Ease        | Simple, Instant, One-click, Effortless          |
| Value       | Free, Save, Bonus, No risk, Worth               |

## Weak vs. Strong Copy Patterns

| Weak                   | Strong                              |
| ---------------------- | ----------------------------------- |
| "Learn more"           | "See how it works"                  |
| "We offer solutions"   | "Cut onboarding time in half"       |
| "Click here"           | "Get your free audit"               |
| "High quality"         | "4.9 stars across 2,400 reviews"    |
| "Our platform is easy" | "Live in 10 minutes, no dev needed" |
